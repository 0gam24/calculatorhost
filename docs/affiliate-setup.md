# 어필리에이트 파트너 가입 가이드

> **운영자(김준혁) 전용 가이드**. 코드 인프라는 모두 구축됨 — 가입 후 환경변수만 입력하면 자동 활성.

## ⚠️ 검증 정책

본 가이드의 모든 프로그램은 **공개 가입 가능 + 개인 운영자 자격**으로 등재. 다음은 본 가이드에서 **제외**한 서비스 (개인용 공개 어필리에이트 없음 — B2B 직접 contact 필요):

| 서비스 | 상태 | 대체 경로 |
|---|---|---|
| 삼쩜삼 (3o3) | 공개 어필리에이트 X | 직접 메일 문의 (제휴 영업) |
| 뱅크샐러드 | 공개 어필리에이트 X | 직접 메일 문의 |
| 핀크 | 공개 어필리에이트 X | 직접 메일 문의 |
| 카드고릴라 | 자체가 어필리에이트 사이트 | 해당 없음 |
| 트래블월렛 | 공개 어필리에이트 X | 직접 메일 문의 |
| 인프런·클래스101 | 강사 모집 위주, 일반 affiliate 미공개 | 직접 contact |

→ 위 서비스를 활용하려면 직접 비즈니스 메일로 제휴 문의 필요. 운영자 판단 후 별도 협의.

## 🎯 즉시 시작 가능 (검증 완료) — 우선순위 순

### 1순위: **쿠팡 파트너스** ⭐ 가장 쉽고 효과 즉시

**왜 1순위?**
- 가입 즉시 승인 (카카오/네이버 간편 로그인)
- 도서·전자책 추천이 모든 가이드와 자연스러운 매칭
- CPS 3% (도서 기준), 24시간 내 클릭한 모든 상품 적용

**가입 절차**:
1. https://partners.coupang.com 접속
2. "파트너스 시작하기" → 카카오/네이버 간편 로그인
3. 정보 입력 → 즉시 승인
4. 대시보드 → "링크 생성" → 추천 도서 검색 → 단축 URL 복사
5. Cloudflare Pages 환경변수:
   - 변수명: `NEXT_PUBLIC_AFFILIATE_COUPANG_URL`
   - 값: 발급받은 단축 URL (예: `https://link.coupang.com/a/XXXXX`)

**활용 가이드**:
- 가이드 끝에 "📚 관련 도서 추천" 섹션
- 검색 키워드 예: "양도세 절세", "재테크 입문", "프리랜서 세금"
- **주의**: 쿠팡파트너스 정책상 "쿠팡파트너스 활동의 일환으로 일정 수수료를 받습니다" 명시 필수 — 본 사이트의 어필리에이트 컴포넌트가 자동 부여

**예상 수익**: 도서 1권당 평균 600~1,500원 (15,000~50,000원 도서 가격 기준 3%). 월 30~50건 클릭·전환 가능 시 월 2~5만원 추정.

### 2순위: **알라딘 TTB** — 도서 클러스터 추가 강화

**왜 2순위?**
- 한국 도서 어필리에이트 표준
- 쿠팡파트너스와 함께 노출하면 사용자 선택권 확대
- TTB 코드 기반이라 추적 안정적

**가입 절차**:
1. https://blog.aladin.co.kr/ttb/ 접속
2. 알라딘 회원 가입 후 TTB 신청서 작성
3. 승인 1~3 영업일
4. 발급된 TTB 코드를 도서 URL 끝에 붙여 사용
5. Cloudflare Pages 환경변수:
   - 변수명: `NEXT_PUBLIC_AFFILIATE_ALADIN_URL`
   - 값: 추천 도서 URL + TTB 코드 (예: `https://www.aladin.co.kr/shop/wproduct.aspx?ItemId=XXXXX&partner=YOUR_TTB`)

**활용 가이드**:
- 쿠팡파트너스와 함께 도서 추천 박스 (사용자 선택권)
- 알라딘 신간 정보 정기 갱신 (월 1회)

### 3순위: **카카오뱅크 친구추천** — 즉시 가능, 5분

**왜 3순위?**
- 별도 가입 절차 없음 (개인 추천 코드)
- 적금·예금·체크카드 계산기와 완벽 매칭
- 추천인·추천받은 사람 모두 5,000원 보너스

**가입 절차**:
1. 카카오뱅크 앱 → 더보기 → **친구초대**
2. **추천 링크** 복사
3. Cloudflare Pages 환경변수:
   - 변수명: `NEXT_PUBLIC_AFFILIATE_KAKAOBANK_URL`
   - 값: 추천 링크 (예: `https://r.kakaobank.com/recommend?code=XXXXX`)

**활용 페이지** (운영자 환경변수 입력 후 추가 PR로 적용):
- `/calculator/savings/` — 적금 이자 계산기
- `/calculator/deposit/` — 정기예금 이자 계산기

**주의**: 개인 추천 코드는 어필리에이트 정책상 "광고" 표시는 의무. 본 사이트 컴포넌트가 자동 부여.

## ⚙️ 환경변수 입력 후 동작 흐름

1. **운영자**: Cloudflare Pages → Settings → Environment variables 에 키 입력
2. **자동**: 다음 배포 시 빌드에 반영
3. **자동**: `<AffiliateRecommendation>` 컴포넌트가 env var 감지 → 자동 노출
4. **자동**: `/affiliate-disclosure` 페이지의 "활성 파트너" 표에 자동 등재

env var 미입력 상태:
- 사이트에 어필리에이트 UI **자동 비표시** (운영자 가입 전 안전 비활성)
- AdSense·SEO·신뢰도 영향 0

## 🛡️ 안전 운영 원칙

1. **회색지대 서비스 추천 금지** — OTT 계정공유 같이 서비스 약관 위반 가능 서비스는 추가 X
2. **한 페이지당 최대 1개** 어필리에이트 추천 (스팸 회피)
3. **광고 라벨 의무** — `<AffiliateRecommendation>` 컴포넌트가 자동 부여
4. **rel="sponsored"** — Google 정책 자동 준수
5. **사용자 결제액 영향 없음 명시** — disclosure 페이지에 명시됨
6. **검증 후 추가** — 새 파트너는 운영자가 실제 가입 가능 여부 확인 후 코드 추가
