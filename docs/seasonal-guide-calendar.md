## Ralph 자동화 로직

### 스크립트 흐름 (scripts/ralph-seasonal-guide.mjs)
```mermaid
1. 현재 월(한국시간) 파악
2. 본 캘린더에서 다음 달(또는 현재 달) 토픽 lookup
3. 발행 여부 확인:
   - src/app/guide/{slug}/page.tsx 존재 여부
4. 미발행 이면:
   - stuck.md에 시그널 등재
   - content-writer 피드 추가
5. 이미 발행 이면:
   - 마지막 수정일 확인
   - 1년 이상 지났으면 갱신 권고 시그널
```

### 의존성
- Node.js 내장 `fs`, `path`, `Date` 만 사용
- 외부 라이브러리 불필요
- 실행: `npm run ralph:seasonal` (또는 GitHub Actions 월 1일 자동 실행)

## 확장 규칙

- **미니 가이드**: 월별 토픽에 포함되지 않는 이슈(세법 개정, 금융상품 변화)는 velocity-trend-monitor로 추적
- **블루오션**: 기존 가이드와 겹치지 않으려면 docs/seo-keyword-map.md §4 참고
- **갱신 사이클**: 기존 가이드는 1년 경과 시 자동 "갱신 권고" 시그널 (스크립트가 감지)
