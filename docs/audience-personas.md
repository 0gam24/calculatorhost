# 타깃 페르소나 & 니치

> **Status**: SSoT — 콘텐츠·기능 결정의 근거
> **Last updated**: 2026-04-24

## 1. 7개 주요 페르소나

### A. 부동산 거래 직전자 🏠
- **나이/상황**: 30-50대, 매매/전세 계약 1-2주 전, 거래액 3-15억
- **검색 쿼리**: "취득세 85m2 1주택 2026", "양도소득세 일시적2주택", "중개수수료 9억"
- **진짜 불안**: "계산 실수로 수백~수천만원 날릴 것"
- **현재 해결책**: 홈택스(어렵) → 블로그(신뢰 의심) → 계산기 사이트
- **Job-to-be-done**: 거래 전 정확한 최종 납부액 확정

### B. 대출 실행 예정자 🏦
- **나이/상황**: 20-40대 신혼·무주택, 주담대·전세대출 1-3개월 전
- **검색 쿼리**: "DSR 40 주담대 한도", "원리금균등 만기일시"
- **불안**: "최대 대출 한도? 월 상환 감당 가능?"
- **Job**: 가능 한도 + 월 부담 시뮬 → 집 선택 범위 확정

### C. 직장인 💼
- **나이/상황**: 20-50대, 연봉협상·이직·퇴사
- **검색 쿼리**: "연봉 5000 실수령액 2026", "세후 월급", "4대보험"
- **불안**: "이직 제안 실수령액? 세금 얼마나 떼?"
- **Job**: 협상 카드로 쓸 정확한 세후 수치

### D. 은퇴 준비층 🧓
- **나이/상황**: 40-60대, 자산 3-20억
- **검색 쿼리**: "은퇴자금 필요액", "국민연금 수령", "상속세 30억"
- **Job**: 은퇴 충당 가능성 수치 확인

### E. 생활 투자자 📈
- **나이/상황**: 20-40대, 주식·적금 병행
- **검색 쿼리**: "주식 물타기", "적금 단리 복리", "정기예금 이자"
- **Job**: 수익률·손익분기 즉석

### F. 임대차 관계자 🏘
- **나이/상황**: 임대인 40-60 / 세입자 20-40
- **검색 쿼리**: "전월세 전환율", "임대료 상승 5%"
- **Job**: 법적 한도 내 계산

### G. 청약 대기자 🎫
- **나이/상황**: 20-40대 무주택
- **검색 쿼리**: "청약가점", "무주택 기간"
- **Job**: 당첨 가능 단지 판단

## 7. 2026 한국 소비 트렌드 반영 (『트렌드 코리아 2026』)

### 7-1. 프라이스 디코딩(Price Decoding)
- **정의**: 최저가 검색 넘어 가격 구조·가치·근거를 분석하여 합리성 판단
- **우리 반영**: 계산기 결과에 **"왜 이 세율/금리가 되는가?"** 해설 섹션 필수
  - 예: "양도세 3억 원 → 왜 과세표준 2.5억 기준 38% 구간 적용되는지"
  - AI 해설 토글이 이를 자동화
- **CPC 연결**: 금융 상품 리뷰 키워드 ($30-40)

### 7-2. 레디코어(Ready Core)
- **정의**: 청년 세대가 생애 이벤트를 노션/엑셀로 통제·계획
- **우리 반영**: FIRE 계산기·복리 시뮬·은퇴자금 등 **생산성 유틸리티** 강화
  - Phase 2 확장: "내 집 마련 마스터 플랜", "은퇴 시뮬레이터"
  - 결과를 **Notion/엑셀 템플릿 다운로드** 기능 (차별화)

### 7-3. 픽셀라이프(Pixel Life)
- **정의**: 관심사의 초세분화, 빠른 변화
- **우리 반영**: 블루오션 니치 전략과 완전 일치 ✅
  - 프리랜서/N잡러/학부모/전업주부 등 극세분화 타깃
  - "20대 직장인 연봉 실수령액" 같은 세그먼트별 가이드 페이지
