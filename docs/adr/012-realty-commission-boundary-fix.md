# ADR-012: 중개수수료 경계값 귀속 수정 (공인중개사법 시행규칙 [별표] 정확화)

**Status**: Accepted  
**Date**: 2026-06-01  
**Agent**: calc-logic-verifier  
**Impact**: `src/lib/finance/realty-commission.ts`, `src/lib/constants/realty-rates-2026.ts`

## 문제

### Off-by-One YMYL 버그
중개수수료 계산 함수 `resolveBracket()`이 경계값을 **낮은 구간**에 귀속시켰습니다.

**예시 (버그)**:
```
매매가 정확히 9억원 → 0.4% 세율 적용 (3600만원)  ❌
```

**법적 근거 위반**: 공인중개사법 시행규칙 §20 [별표]
```
"9억 이상~12억 미만: 0.5%"
→ 9억은 "이상"이므로 이 구간에 포함
→ 올바른 계산: 9억 × 0.5% = 450만원  ✓
```

### 영향
- **사용자 손실**: 경계값 거래에서 수십만~수백만원 중개수수료 오류
- **YMYL 위반**: 금융 계산 정확성 핵심 (Search Console 순위 하락 위험)
- **법적 책임**: 부동산 중개수수료는 법정 상한액 기준이므로, 오류 계산은 신뢰성 문제

## 해결책

### 1. upperBound 의미 재정의
**이전**: `<=` 연산자로 경계값을 현재 구간에 포함  
**변경**: `<=` 유지하되, upperBound 값을 **"미만"** 경계로 설정

```typescript
// 이전
{ upperBound: 900_000_000, rate: 0.004 }  // "9억 이상은 0.4%"라는 오독
// 현재 (올바름)
{ upperBound: 899_999_999, rate: 0.004 }  // "9억 미만 0.4%" → 9억은 다음 구간

// 다음 구간
{ upperBound: 1_200_000_000, rate: 0.005 } // 9억 ≤ amount ≤ 12억: 0.5%
```

### 2. 배열 순서 원칙
배열을 순서대로 탐색하면서 `amount <= bracket.upperBound` 첫 매칭을 반환:

```
input: 900,000,000
└─ bracket[0] upperBound=50M:       900M <= 50M? NO
└─ bracket[1] upperBound=200M:      900M <= 200M? NO
└─ bracket[2] upperBound=899M:      900M <= 899M? NO   ← "미만" 경계
└─ bracket[3] upperBound=1200M:     900M <= 1200M? YES ✓ → 0.5% (올바름)
```

### 3. 법규 해석 명확화
**공인중개사법 시행규칙 §20 [별표] 구간 정의**:

**주택 매매**:
```
~5,000만원                 0.6% (한도 25만)        [0 ≤ amount ≤ 50M]
5,000만 초과~2억          0.5% (한도 80만)        [50M < amount ≤ 200M]
2억 초과~9억 미만         0.4%                    [200M < amount < 900M]
9억 이상~12억 미만        0.5%                    [900M ≤ amount < 1200M]
12억 이상~15억 미만       0.6%                    [1200M ≤ amount < 1500M]
15억 이상                 0.7%                    [1500M ≤ amount]
```

**주택 임대차**:
```
~5,000만원                 0.5% (한도 20만)        [0 ≤ amount ≤ 50M]
5,000만 초과~1억 미만     0.4% (한도 30만)        [50M < amount < 100M]
1억 이상~6억 미만         0.3%                    [100M ≤ amount < 600M]
6억 이상~12억 미만        0.4%                    [600M ≤ amount < 1200M]
12억 이상~15억 미만       0.5%                    [1200M ≤ amount < 1500M]
15억 이상                 0.6%                    [1500M ≤ amount]
```
