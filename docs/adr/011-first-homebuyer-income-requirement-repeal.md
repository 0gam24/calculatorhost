# ADR-011: 생애최초 주택 취득세 감면 소득요건 폐지 반영

**Status**: COMPLETED
**Date**: 2026-06-01
**Verifier**: calc-logic-verifier (TDD: RED→GREEN, 953 tests PASS)

---

## 1. 배경

### 법령 개정
지방세특례제한법(이하 "지특법")의 **생애최초 주택 취득세 감면 규정**에서 소득요건이 폐지됨:

- **폐지 시점**: 2022년 6월 21일
- **개정 조항**: §36의2 → **§36의3** (조항 번호 변경)
- **폐지된 조건**: 부부합산 소득 7,000만원 이하
- **유지된 조건**: 무주택 세대, 주택가액 12억원 이하, 1주택 매매

### 현행 상태 (2026년)
- 소득요건 **완전 폐지**
- 무주택 세대주만 확인하면 감면 가능
- 최대 감면액: 200만원 (변경 없음)

## 3. 해결책 (TDD 적용)

### 3-1. RED 단계 (테스트 기존 상태)
기존 취득세 테스트는 **이미 소득 검증 불필요**:
- `calculateAcquisitionTax()` 함수는 `firstHomeBuyerDiscount: boolean` 플래그만 사용
- 호출처에서 소득 게이트 로직이 없음
- 테스트도 소득요건 미적용 상태

→ 테스트 변경 필요 없음 (이미 현행법 준수)

### 3-2. GREEN 단계 (코드 수정)

#### 수정 1: 상수 정리
**파일**: `src/lib/constants/tax-rates-2026.ts` (L200-207)

```typescript
// ✅ 수정된 코드
// ─── 생애최초 주택 감면 ───
/** 생애최초 주택 감면 한도 (원) — 지특법 §36의3 */
firstHomeBuyerMaxDiscount: 2_000_000,
/** 생애최초 감면 주택가액 제한 — 지특법 §36의3 (소득요건은 2022.6.21자로 폐지됨, 무주택 세대만 필요) */
firstHomeBuyerMaxPrice: 1_200_000_000,
```

**변경 사항**:
- `firstHomeBuyerMaxIncome` 상수 제거 (폐지된 요건)
- 조항 번호 `§36의2` → `§36의3`
- 주석 추가: 소득요건 폐지 명시

#### 수정 2: 페이지 FAQ 갱신
**파일**: `src/app/calculator/acquisition-tax/page.tsx` (L62-65)

**이전**:
> 지특법 §36의2에 따라 생애최초 주택 구매 시 최대 200만 원의 취득세 감면을 받을 수 있습니다. 단, 부부합산 소득 7,000만 원 이하, 주택가액 12억 원 이하, 1주택 매매 조건을 만족해야 합니다.

**이후**:
> 지특법 §36의3에 따라 생애최초 주택 구매 시 최대 200만 원의 취득세 감면을 받을 수 있습니다. 조건은 무주택 세대(본인·배우자 모두 무주택)이고, 주택가액 12억 원 이하, 1주택 매매이어야 합니다(소득요건은 2022.6.21자로 폐지됨). 매매 잔금일로부터 60일 이내에 위택스에서 신청하세요.

#### 수정 3: 주의사항 갱신
**파일**: `src/app/calculator/acquisition-tax/page.tsx` (L335-338)

**이전**:

**이후**:
> 생애최초 주택 감면은 무주택 세대(본인·배우자 무주택), 주택가액 12억 원 이하, 1주택 매매 조건을 만족해야 적용됩니다(지특법 §36의3, 2022.6.21자 소득요건 폐지).

#### 수정 4: 용어사전 갱신
**파일**: `src/app/calculator/acquisition-tax/page.tsx` (L192-194)

**이후**:
