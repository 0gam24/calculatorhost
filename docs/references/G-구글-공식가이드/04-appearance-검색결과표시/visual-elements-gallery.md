# Google 검색 시각적 요소 갤러리

시각적 요소는 Google 검색 결과 페이지의 기본 요소로, 사용자가 인지하거나 상호작용할 수 있습니다. 시각적 요소 갤러리는 Google 웹 검색의 가장 일반적인 UI 요소인 시각적 요소를 그림을 통해 설명하는 가이드입니다. 시각적 요소가 어떻게 생겼는지, 이름은 무엇인지와 더불어 각 요소에 맞게 웹사이트를 최적화할 수 있는지 어떤지 설명합니다.

## Google 검색결과 페이지 분석

Google 검색 결과 페이지에는 다양한 유형의 검색 결과 시각적 요소가 포함되어 있으며, 각 검색 결과마다 고유한 하위 시각적 요소 세트가 사용됩니다. 예를 들어 *텍스트 검색 결과*는 그 자체로 시각적인 요소이며 속성, 제목 링크, 스니펫과 같은 다양한 하위 시각적 요소를 갖습니다.

시각적 요소가 표시되는 방식은 시간이 지남에 따라 달라질 수 있으며, 특정 검색 결과는 사용자가 데스크톱 컴퓨터와 휴대전화 중 무엇을 사용하고 있는지, 어떤 국가에 살고 있는지, 검색어의 언어는 무엇인지 및 기타 여러 가지 요인에 따라 다르게 표시될 수 있습니다. 다음은 Google 검색에 가장 일반적으로 표시되는 검색 결과 시각적 요소 유형입니다.

텍스트 검색 결과, 리치 결과, 이미지 검색 결과, 동영상 검색 결과, 탐색 분석 기능 등 가장 일반적인 유형의 검색 결과를 언급하는 Google 검색결과 페이지 그림

## 기여 분석

## 텍스트 검색결과

*텍스트 검색 결과*(이전의 '웹 검색 결과' 또는 '일반 파란색 링크')는 페이지의 텍스트 콘텐츠를 기반으로 하는 Google 검색 결과입니다. 여기에는 기여 분석, 제목 링크, 스니펫과 같은 시각적 요소가 포함됩니다.

텍스트 검색 결과에는 리치 속성이나 사이트링크 그룹과 같은 시각적 요소도 추가적으로 포함될 수 있습니다. 특정 텍스트 결과는 사용 중인 기기, 검색한 내용, 사용하는 언어 등 다양한 요소에 따라 서로 다르게 표시될 수도 있음을 기억해 두시기 바랍니다. 포함될 수 있는 모든 시각적 요소가 있는 텍스트 검색결과는 표시되지 않습니다.

텍스트 검색결과의 특정 시각적 요소에 라벨이 지정되어 있고, 링크(이를 제어하는 방법에 관한 자세한 정보 제공)가 표시된 Google 검색 텍스트 검색결과를 보여주는 그림

### 텍스트 검색결과 이미지

*텍스트 검색결과 이미지*는 특정 웹페이지에서 검색어와 관련성이 가장 높은 이미지입니다. 사용자가 텍스트 검색 결과 이미지를 탭하면 이미지가 삽입된 웹페이지로 이동합니다. 이미지를 검색하는 쿼리에는 텍스트 검색 결과 이미지가 표시될 가능성이 높습니다.

텍스트 검색결과 이미지에 맞게 최적화하려면 [이미지 검색엔진 최적화 권장사항](https://developers.google.com/search/docs/appearance/google-images?hl=ko)을 따르세요.

어떤 요소가 텍스트 검색결과 이미지(빨간 튤립의 작은 미리보기 이미지)인지 보여주는 Google 검색 텍스트 검색결과 그림


텍스트 검색결과 이미지

### 리치 속성

*리치 속성*은 리뷰 별점 및 레시피 정보와 같이 웹페이지에 관한 추가 정보이며 1줄 이상 표시됩니다. 이 정보는 일반적으로 사용자가 제공한 [구조화된 데이터](https://developers.google.com/search/docs/appearance/structured-data/intro-structured-data?hl=ko)를 기반으로 표시됩니다.

어떤 요소가 리치 속성(리뷰 별점 정보)인지를 나타내는 Google 검색 텍스트 검색결과 그림

## 이미지 검색결과

*이미지 검색결과*는 웹페이지에 삽입된 이미지를 기반으로 표시되는 검색결과입니다.
이미지를 찾는 검색어에 이미지 검색결과가 표시될 가능성이 높습니다. 이미지 검색 결과에 맞게 이미지를 최적화하려면 [이미지 검색엔진 최적화 권장사항](https://developers.google.com/search/docs/appearance/google-images?hl=ko)을 따르세요.

기여 분석 및 이미지 썸네일 시각적 요소가 있으며 Google 검색에 표시된 토끼 이미지 검색결과 4개를 보여주는 그림

## 탐색 분석 기능

탐색 분석 기능은 검색자가 원래 검색어와 관련된 다른 질문이나 검색어('다른 사용자가 검색한 항목'이라고도 함)를 탐색하는 데 도움이 됩니다. 여기에 표시되는 내용을 제어할 수는 없지만, 사이트에 관해 작성할 만한 주제를 생각해 볼 때 관련 검색어를 살펴보는 것이 도움이 될 수 있습니다.

### 관련 검색어 그룹

![관련 검색어 그룹이 Google 검색에서 어떻게 표시되는지 보여주는 그림으로, 다른 사용자가 검색한 관련 항목이 표시됨](https://developers.google.com/static/search/docs/images/related-searches-group.png?hl=ko)

*관련 검색어 그룹*은 다른 사용자가 수행한 관련 검색어 클러스터입니다. 사용자가 관련 검색어를 탭하거나 클릭하면 다른 검색 결과 페이지로 이동합니다. 이러한 검색어는 최초 검색어 및 사용자가 검색한 내용을 바탕으로 자동 생성됩니다.

### 관련 질문 그룹

![Google 검색에서 관련 질문 그룹이 어떻게 표시되는지 보여주는 그림으로, 사용자가 처음 검색한 내용과 관련이 있는 일련의 질문이 표시됨](https://developers.google.com/static/search/docs/images/related-questions-group.png?hl=ko)

*관련 질문 그룹*은 사용자가 처음에 검색한 내용과 관련이 있는 질문의 클러스터입니다('다른 사용자가 검색한 항목'이라고도 함). 사용자가 질문을 펼치면 추천 스니펫이 표시됩니다.
[추천 스니펫을 관리](https://developers.google.com/search/docs/appearance/featured-snippets?hl=ko)하는 방법을 알아보세요.
