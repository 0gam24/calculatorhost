# Google의 웹 스토리 콘텐츠 정책

[웹 스토리](https://developers.google.com/search/docs/guides/enable-web-stories?hl=ko)가 Google 디스커버 및 Google 검색에 단일 결과로 표시될 수 있도록 하려면 [디스커버 정책](https://support.google.com/websearch/answer/9982767?hl=ko) 및 [Google 검색 Essentials](https://developers.google.com/search/docs/essentials?hl=ko), [Google 검색의 전반적인 정책](https://support.google.com/websearch/answer/10622781?hl=ko)을 준수해야 합니다.
또한 웹 스토리가 Google 검색의 더 풍부한 환경(예: 캐러셀 뷰)에 표시될 수 있도록 하려면 웹 스토리가 [검색 결과 기능 콘텐츠 정책](https://support.google.com/websearch/answer/10622781?hl=ko)과 다음 추가 콘텐츠 정책을 준수해야 합니다. 웹 스토리 콘텐츠 정책을 심각하게 위반하는 경우 사이트가 Google의 더 풍부한 환경에 영구적으로 표시되지 않을 수 있습니다.

### 저작권 보호 콘텐츠

타인의 저작권을 침해하는 웹 스토리는 허용되지 않습니다. 웹 스토리는 원본 저작물을 반영하기 위한 것이므로 허가를 받지 않은 한 타인의 저작권 보호 저작물이 포함된 웹 스토리는 허용되지 않습니다. Google은 웹 스토리가 전반적인 Google 서비스에 표시되도록 권리를 취득하는 것과 관련하여 어떠한 의무 또는 책임도 없습니다. 웹 스토리가 타인의 저작권을 침해하는 경우 Google은 웹 스토리가 표시되지 않도록 차단할 수 있습니다. 자세한 내용은 [Google의 저작권 절차](https://support.google.com/legal/troubleshooter/1114905?hl=ko)를 참고하세요.

### 텍스트가 많은 웹 스토리

텍스트가 많은 웹 스토리는 허용되지 않습니다. 대다수 [페이지](https://amp.dev/documentation/components/amp-story-page/?format=stories)에 180자 이상의 텍스트가 있는 웹 스토리는 표시되지 않을 수 있습니다. 가능하다면 짧은 동영상(페이지당 60초 미만)을 사용하는 것이 좋습니다.

### 품질이 낮은 애셋

사용자의 경험에 부정적인 영향을 미치는 수준까지 확대되거나 모자이크 처리된 이미지 및 동영상 애셋이 포함된 웹 스토리는 허용되지 않습니다.

### 부족한 이야기

페이지 간 결합 테마 또는 이야기 구조가 누락되어 있는 웹 스토리는 허용되지 않습니다.

### 불완전한 스토리

불완전하거나 사용자가 다른 웹사이트 또는 앱으로 연결되는 링크를 클릭해야만 필수 정보를 얻을 수 있는 웹 스토리는 허용되지 않습니다.

### 지나친 광고

서비스나 제품을 광고하는 것이 유일한 목표이고 특히 웹 스토리를 소비하는 사용자에게서 직접 이익을 얻을 수 있는 웹 스토리는 허용되지 않습니다. 제휴 마케팅 링크는 웹 스토리의 일부분으로만 제한된다면 허용됩니다. 디스플레이 광고는 [스토리 광고 가이드라인](https://amp.dev/documentation/guides-and-tutorials/develop/advertise_amp_stories/?format=websites)을 따라 배치할 수 있습니다.
[Google 제휴 프로그램](https://developers.google.com/search/docs/essentials/spam-policies?hl=ko#thin-affiliate-pages)은 [Google 웹 검색의 스팸 정책](https://developers.google.com/search/docs/essentials/spam-policies?hl=ko)에 따라 지원됩니다.
