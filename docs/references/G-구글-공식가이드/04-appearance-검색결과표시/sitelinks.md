# 사이트링크

사이트링크는 텍스트 검색결과에 함께 클러스터링된 동일한 도메인의 링크입니다.
Google 시스템은 사이트의 링크 구조를 분석하여 시간 단축을 위한 지름길을 찾고, 사용자가 원하는 정보를 신속하게 찾을 수 있도록 합니다.

Google에서는 사용자에게 유용하다고 판단되는 검색결과의 사이트링크만 표시합니다. 유용한 사이트링크를 찾아 주는 Google 알고리즘이 사이트 구조에서 허용되지 않거나 사이트의 사이트링크가 검색어와 관련 없을 경우 사이트링크를 표시하지 않습니다.

## 사이트링크 권장사항

현재 사이트링크는 자동화되어 있습니다. Google은 사이트링크 알고리즘을 지속적으로 개선하고, 향후 사이트 소유자의 의견을 반영할 수 있도록 노력 중입니다. 하지만 사이트링크의 품질을 개선하기 위해 따를 수 있는 권장사항이 있습니다.

* 페이지 제목 및 표제에 사용하는 텍스트는 유익하고, 관련성이 높으며, 간결해야 합니다.
* 사용자가 쉽게 탐색할 수 있는 논리적인 사이트 구조를 만들고 다른 관련 페이지에서 중요한 페이지로 연결되도록 합니다.
* 내부 링크의 앵커 텍스트는 간결해야 하며 내부 링크가 가리키는 페이지와 관련이 있어야 합니다.
* 콘텐츠가 반복해서 표시되지 않도록 합니다.

사이트링크를 삭제해야 하는 경우 사이트에서 페이지를 삭제하거나 [`noindex`](https://developers.google.com/search/docs/crawling-indexing/block-indexing?hl=ko)를 사용해 보세요.
