# 새 전자상거래 웹사이트 출시 방법

상거래 사이트가 출시될 때 Google이 찾을 수 있도록 하려면 다음과 같은 방법으로 새 사이트를 Google에 등록하는 것이 좋습니다.

1. [Google에 사이트 소유권을 인증합니다](https://support.google.com/webmasters/answer/9008080?hl=ko).
2. [Google에 사이트 색인 생성을 요청합니다](https://developers.google.com/search/docs/crawling-indexing/ask-google-to-recrawl?hl=ko).

   * **URL 수가 적은 경우:**
     [URL 검사 도구](https://support.google.com/webmasters/answer/9012289?hl=ko)를 사용합니다.
   * **URL 수가 많은 경우:** [사이트맵](https://developers.google.com/search/docs/crawling-indexing/sitemaps/build-sitemap?hl=ko)을 제공합니다.
3. [Google 검색에서 사이트의 색인이 생성되는 과정을 추적합니다](https://support.google.com/webmasters/answer/7440203?hl=ko).
   Google 판매자 센터에서는 Google 검색 색인으로 콘텐츠를 확인하고 비교 검토할 수 있습니다. 잘못된 인증 경고를 방지하려면 Google 검색에 사이트의 색인이 생성되고 제품을 구매할 수 있어야 합니다. [페이지 색인 생성 보고서](https://support.google.com/webmasters/answer/7440203?hl=ko)를 통해 색인 생성 진행 상황을 확인할 수 있습니다.
4. 오프라인 상점도 있다면 [Google에 비즈니스 세부정보를 설정](https://developers.google.com/search/docs/appearance/establish-business-details?hl=ko)합니다.
5. [Google 판매자 센터에 가입](https://support.google.com/merchants/answer/188924?ref_topic=7294166&hl=ko)하여 제품 설명 및 특가 상품 소개와 같은 추가 정보를 제공합니다.

**참고**:
사이트가 Google 쇼핑 탭과 같은 환경에 표시되도록 하려면
[Google 판매자 센터에 가입](https://support.google.com/merchants/answer/188924?ref_topic=7294166&hl=ko)해야 합니다.
가입하면 Google 이미지에서 이미지에 제품 배지를 추가하는 등 기타 풍부한 기능도 제품에 사용할 수 있습니다.

사이트를 출시할 때 마케팅을 고려해야 할 수도 있습니다. 출시 즉시 Google에서 사이트를 이용할 수 있도록 하는 다양한 접근 방식의 장단점을 자세히 알아보세요.

* [완전 공개](#grand-reveal): 전체 사이트를 일반 대중과 Google에 동시에 공개합니다.
* [홈페이지 출시](#home page-launch): 처음에는 사이트의 홈페이지만 일반 대중과 Google에 공개합니다.
* [제품 재고 없이 출시](#products-unavailable):
  제품을 '재고 없음'으로 표시하고 전체 사이트를 일반 대중과 Google에 출시합니다.
* [제한적 출시](#soft-launch): 전체 사이트를 조기에 출시하고 '공식' 출시를 위한 마케팅 이벤트는 별도로 진행합니다.

## 완전 공개

한 가지 접근 방식으로 출시 전에는 일반 대중과 검색엔진에서 전체 전자상거래 사이트에 액세스하지 못하도록 합니다. 예를 들어 비밀번호가 있어야 사이트의 페이지에 액세스할 수 있게 하면 Google에서는 사이트에 액세스하지 못하지만, 테스터는 액세스하여 일반 대중에게 공개되기 전에 사이트가 잘 작동하는지 확인할 수 있습니다. 그런 다음 다른 마케팅 활동과 조율하여 원하는 시점에 사이트를 '완전 공개'합니다.

**작업**: 출시 직후에 [전자상거래 사이트 출시](#steps) 단계를 따르세요.
이렇게 하면 Google 검색에 사이트가 공개되기까지의 지연을 최소화할 수 있습니다.

|  |  |
| --- | --- |
| **장점** 출시 전에 콘텐츠가 알려지지 않으며, 마케팅 캠페인에 이 점이 중요하게 작용할 수 있습니다. | **단점** 사이트가 Google 검색결과와 Google 쇼핑 탭에 표시되는 데 시간이 더 오래 걸릴 수 있습니다. |

## 홈페이지 출시

홈페이지만 Google에 공개한 상태로 사이트를 출시할 수 있습니다. 매장을 설명하는 문구를 담은 홈페이지를 사용해 전자상거래 웹사이트가 곧 출시될 예정임을 알립니다.

**작업**: 미리 [사이트 소유권을 인증](https://support.google.com/webmasters/answer/9008080?hl=ko)해 두고, 사이트에서 제품을 구매할 수 있게 되면 [나머지 단계](#steps)를 진행합니다.

|  |  |
| --- | --- |
| **장점** 이 접근 방식을 사용하면 매장을 등록하고 도메인 이름의 인지도를 높이며 다른 사용자가 내 사이트에 연결하도록 허용할 수 있습니다. 또한 출시 시점에는 아직 제품을 구매할 수 없더라도 Google 검색에서 매장 이름을 찾을 수 있게 됩니다. | **단점** 전체 사이트가 출시되고 Google에서 크롤링하여 색인을 생성할 수 있을 때까지는 Google 검색과 쇼핑 탭에서 자세한 정보를 확인할 수 없습니다. |

## 제품 재고 없이 출시

전체 사이트를 출시하되, 주문을 처리할 준비가 될 때까지는 구매 기능을 사용 중지할 수 있습니다. 예를 들어 제품을 '재고 없음'으로 표시할 수 있습니다. 이렇게 하면 제품을 포함한 모든 사이트 콘텐츠의 색인이 생성됩니다. 각 페이지에 사이트의 공식 출시일을 알리는 메시지를 포함하는 것이 좋습니다.

**주의**: 사용자가 제품을 구매하지 못하게 하기 위해 '장바구니에 추가' 기능을 사용 중지하면 안 됩니다.
Google에서 세금과 배송비가 포함된 최종 가격 세부정보를 확인할 때 '장바구니에 추가'를 사용할 수 있습니다.

**작업**: [전자상거래 사이트 출시 단계를 모두 실행](#steps)하되, [`excluded_destination` 속성](https://support.google.com/merchants/answer/6324486?hl=ko)을 사용하여 재고를 구매 불가로 표시한 상태로 제품 데이터를 Google 판매자 센터에 제공합니다.
이렇게 하면 Google 쇼핑과 같은 대상 유형에서 제품을 구매 가능으로 표시하지 않고도 제품 데이터 로드 확인 문제를 조기에 식별할 수 있습니다.

|  |  |
| --- | --- |
| **장점** 사이트를 출시하는 즉시 콘텐츠의 색인이 생성됩니다. | **단점** 고객에게는 사이트가 공개된 것처럼 보이지만 주문을 할 수는 없습니다. 구매 거래를 완료하지 못한 고객에게 불만이 쌓이지 않게 하려면 쇼핑객에게 사이트가 아직 완전히 출시되지 않았다고 명확히 밝히는 것이 좋습니다. |

## 제한적 출시

또 다른 접근 방식은 운영 준비가 완료되는 즉시 사이트의 모든 기능을 사용 설정하는 것입니다.
그런 다음 나중에 출시 마케팅을 진행할 수 있습니다. 공식 출시 전에 사이트를 둘러본 쇼핑객이 특별 대우를 받았다고 느끼게 만드는 '미리보기' 이벤트로 할 수도 있습니다.

**작업**: 사이트가 공개되는 즉시 [전자상거래 사이트 출시 단계를 모두 실행](#steps)합니다.

|  |  |
| --- | --- |
| **장점** 이 접근 방식은 단순함이 장점입니다. 또한 사이트가 완전히 공개되므로 트래픽이 적은 상태에서 실제 사용자가 테스트할 수 있습니다. | **단점** 사용자가 소셜 미디어에서 사이트를 홍보하거나 언급하면 계획된 출시일 이전에 사이트에 대한 관심이 높아질 수 있습니다. |
