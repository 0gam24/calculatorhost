# 사이트에서 악용 방지 및 모니터링

다음 주제에서는 사이트에서 악용을 방지하고 모니터링하는 방법을 설명합니다.

| 주제 | |
| --- | --- |
| [사이트 또는 플랫폼에서 사용자 생성 스팸 방지하기](https://developers.google.com/search/docs/monitor-debug/prevent-abuse?hl=ko) | 스팸 발송자는 종종 열린 댓글 양식과 기타 사용자 제작 콘텐츠 입력을 활용하여 무방비 상태의 피해 사이트에서 스팸 콘텐츠를 생성합니다. 사이트 또는 플랫폼에서 악용을 방지하고 모니터링하는 방법을 알아보세요. |
| [멀웨어 및 원치 않는 소프트웨어](https://developers.google.com/search/docs/monitor-debug/security/malware?hl=ko) | 멀웨어 및 원치 않는 소프트웨어의 정의, 원치 않는 소프트웨어 배포를 방지하기 위한 가이드라인, 원치 않는 소프트웨어와 관련된 문제를 해결하는 방법을 알아보세요. |
| [멀웨어 감염 방지하기](https://developers.google.com/search/docs/monitor-debug/security/prevent-malware?hl=ko) | 이 문서에는 멀웨어 감염을 방지하기 위한 도움말이 포함되어 있습니다. |
| [소셜 엔지니어링(피싱 및 사기성 사이트)](https://developers.google.com/search/docs/monitor-debug/security/social-engineering?hl=ko) | 소셜 엔지니어링의 정의와 해결 방법을 알아보세요. |
| [Google 세이프 브라우징 반복 위반자 관련 정책](https://developers.google.com/search/docs/monitor-debug/security/safe-browsing-repeat-offenders?hl=ko) | Google 세이프 브라우징은 사용자에게 위험한 사이트나 다운로드 파일 관련 경고를 표시하여 사용자를 보호합니다. |
