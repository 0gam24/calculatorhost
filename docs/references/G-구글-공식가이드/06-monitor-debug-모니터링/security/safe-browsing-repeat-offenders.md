# Google 세이프 브라우징 반복 위반자 관련 정책

Google 세이프 브라우징은 위험한 사이트나 다운로드 파일에 경고를 표시하여 사용자를 보호합니다. 또한, 웹사이트가 악의적인 행위자에 의해 해킹당했을 때 웹사이트 소유자에게 이를 알리고 문제 진단 및 해결을 도와 방문자가 안전하게 웹사이트를 이용할 수 있게 합니다.

짧은 기간 내에 정책 준수 행위와 정책 위반 행위를 번갈아 가며 반복하는 사이트는 반복 위반자로 분류됩니다.

사이트가 반복 위반자로 지정되면 Search Console에 등록된 웹사이트 소유자 이메일 주소로 알림 이메일이 전송됩니다. 세이프 브라우징에서 사이트를 반복 위반자로 지정하면 웹사이트 소유자는 Search Console을 통해 추가 검토를 요청할 수 없게 됩니다. 반복 위반자 상태는 30일 동안 지속되며 웹사이트 소유자는 30일 후에 검토를 요청할 수 있습니다.
