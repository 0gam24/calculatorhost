# IndexNow 소개

서치 어드바이저는 [IndexNow protocol](https://www.indexnow.org/ko_kr/)을 지원합니다.

IndexNow를 통해 새로 만들어진 페이지나 업데이트, 삭제가 발생한 페이지를 네이버 검색에 알릴 수 있습니다. IndexNow를 통해 웹사이트의 변화를 알리면, 검색로봇이 웹사이트를 방문하도록 기다리지 않아도 됩니다. 또한, 갱신된 페이지 정보가 다른 검색사들에게도 공유됩니다.

주의할 점은 IndexNow protocol 은 검색엔진에 웹사이트의 변화를 빨리 알리는 역할을 하지만, 웹페이지의 색인을 보장하지는 않습니다.

IndexNow 프로토콜을 사용하기 위해서는 HTTP 와 Json 데이터를 이용한 API를 이용해야 합니다.
