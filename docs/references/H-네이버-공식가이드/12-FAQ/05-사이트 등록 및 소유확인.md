# 사이트 등록 및 소유확인

네이버 웹 검색은 웹 수집 로봇이 자동으로 수집한 사이트 정보를 반영하여 노출하고 있습니다. 따라서 검색 등록 절차는 존재하지 않습니다. [검색엔진 최적화](<../01-검색엔진 최적화 기초/01-검색엔진 최적화의 목적.md>) 가이드를 참고하여 검색로봇이 웹사이트를 잘 수집할 수 있도록 조처를 하는 것이 우선입니다.

네이버 웹마스터도구에 사이트를 등록하지 않아도 네이버 검색 결과에 사이트 및 웹문서들이 반영됩니다. 하지만, 네이버 웹마스터도구에 사이트 등록후 [소유확인](https://help.naver.com/support/contents/contents.help?serviceNo=14882&categoryNo=15032)절차를 진행하면, 사이트내 콘텐츠의 수집, 색인, 노출정보를 리포트를 통해서 파악 할 수 있습니다. 소유확인 이후에는 웹마스터도구의 리포트를 활용하여 내 사이트의 검색 연동 상황을 확인할 수 있으므로 사이트 운영에도 많은 도움이 될 수 있습니다.

소유확인은 자바스크립트 실행을 지원하지 않으니 마크업 확인은 최신 웹브라우저의 URL 입력란에 view-source:사이트\_URL 을 입력하거나 페이지 소스보기를 활용하시기 바랍니다.

서치어드바이저에서 소유 확인용 메타태그 및 HTML 검증 시 주의해야 할 점은 아래와 같습니다.

- 서치어드바이저에서 소유확인용 메타태그 검증 시 **자바스크립트, meta refresh, cookie** 를 활용한 브라우저단의 리다이렉션은 지원하지 않습니다. 리다이렉트가 필요하다면 웹서버에서 처리되는 [http 리다이렉트 (301, 302)](<../01-검색엔진 최적화 기초/08-HTTP 규약\(Protocol\).md>)로 처리해주세요.

```
# another.html 에 존재하는 소유확인용 메타태그 확인 불가
<head>
...
<meta http-equiv="refresh" content="0;url=another.html">
...
</head>
```

```
# another.html 에 존재하는 소유확인용 메타태그 확인 불가
<html>
...
<script lang=javascript>
window.location.href = "./another.html"
</script>
...
</html>
```

```
# 소유확인용 메타태그 확인 불가
<html>
...
<script lang=javascript>
document.cookie = 'sample_key=sample_value'
window.location.reload();
</script>
...
</html>
```

- 소유확인용 메타태그 정보가 html frame 태그 내에 존재 시 검증에서 제외 처리합니다. frame 태그가 아닌 html head 태그 내에 기입해 주세요.

```
...
<FRAME SRC="http://another.domain.com">
...
```

- 소유확인용 메타태그 정보가 html body 태그 내에 존재 시 검증에서 제외 처리합니다. html head 태그 내에 기입해 주세요.

```
# 소유확인용 메타태그가 body 태그내에 존재하므로 확인 불가
<html>
<head>
<title>예제</title>
</head>
<body>
<meta name="naver-site-verification" content="...."/>
</body>
</html>
```
