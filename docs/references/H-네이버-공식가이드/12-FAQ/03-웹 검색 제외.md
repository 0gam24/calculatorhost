# 웹 검색 제외

### 네이버 검색의 사이트 제외 방침

네이버는 법률 위반, 저작권 위반과 같은 정책위반의 경우에만 네이버 검색 결과에서 삭제 할수 있습니다. 사이트가 정상적으로 접속되고 있다면 네이버에서 임의로 삭제할 수 없는 점 양해 부탁드립니다.

웹페이지의 차단 또는 삭제는 대개 웹사이트를 제어하는 관리자 또는 소유자를 통해 조치가 되어야 합니다. 콘텐츠를 삭제할 권한이 있는 오너에게 삭제요청을 해야하는 이유는 검색결과에서 삭제되더라도 콘텐츠 자체가 삭제된것이 아니기 때문에 관련 페이지는 여전히 남아 사이트 링크 URL, 소셜미디어와 다른 검색엔진등에 표시될 가능성이 높기 때문입니다.

법적 근거가 될수 있는 명예훼손 등에 의해 게시물 삭제에 대한 법원 명령을 가지고 있는 경우 [고객센터](https://help.naver.com/support/alias/webmastertool/webmastertool_19.naver)에 문의해주세요.

### Q. 제가 소유하는 사이트를 네이버 검색결과에 제외하고 싶습니다.

사이트를 폐쇄하는 것이 아닌, 네이버 검색에서만 제외 싶으시다면 아래와 같은 작업중 하나 또는 이상을 진행해야합니다.

#### 1\. 웹페이지 내 NOINDEX 메타태그를 삽입합니다

`<head><meta name="robots" content="noindex"></head>`

모든 문서에 명시해야하는 번거로움이 있지만 위 메타태그를 추가하면  
로봇은 해당 문서는 검색 결과에 노출을 원하지 않는다는 것을 알 수 있게 됩니다.

단, 단순히 다른 URL로 리다이렉트 되는 페이지인 경우에는, 이 설정만으로 URL이 제외되지 않을 수도 있습니다.  
이 경우에는 robots.txt도 추가 설정하거나, 리다이렉트 타겟이 된 페이지들에도 noindex 메타 설정이나 완전 삭제 처리가 필요합니다.

#### 2\. 웹페이지를 완전 삭제 또는 에러 페이지 반환

웹페이지가 더이상 필요가 없다고 생각될 경우, 웹서버에서 403 (Forbidden) 또는 404 (Not Found)과 같이 알맞은[HTTP 응답 코드](<../01-검색엔진 최적화 기초/08-HTTP 규약\(Protocol\).md>)를 반환하도록 조치합니다.  
유사하게 에러 페이지로 리다이렉트 시키거나, 인증이 필요한 페이지임을 안내하는 방법도 가능합니다. 다만, 페이지 구성에 따라 정상적인 응답으로 로봇이 오인하는 경우도 있기 때문에 주의가 필요합니다.

특히 내용물을 그대로 둔 상태에서, javascript나 CSS를 이용하여 화면을 덮는 형태 등으로 에러 메시지나 로그인 등을 불러오게 하는 등의 기법을 사용하는 경우에는  
로봇의 페이지 해석 과정에서 이 정보가 제대로 인식되지 않을 수 있습니다.

#### `주의: robots.txt를 먼저 설정하거나 서버의 네트워크를 먼저 정지하거나 블럭하지 마세요`

robots.txt는 검색 로봇의 추가적인 접근을 차단하는 설정으로써, 해당 문서나 사이트가 인터넷 상에서 사라졌는지와는 무관합니다.

즉, 해당 사이트나 문서가 삭제되었는지를 확인하기 위해서는 로봇이 그 사이트를 방문해서 확인해야 합니다만.  
robots.txt 등이 앞에 설정되면, 이를 확인하기 위한 접근 조차도 차단되기 때문에, 원하시는 동작이 되지 않을 수 있습니다.

또한, 서버의 네트워크를 무조건 정지하거나 네트워크 레벨에서 블럭하면, 위와 마찬가지로, 로봇 입장에서는 이 서버가 계속 운영중이나 일시적 오류인지 등을 파악할 수 없으므로,  
원하는 바의 반영이 불가능할 수 있습니다.

만일 향후에도 검색 노출을 원하지 않는 경우에는, 먼저 가이드에 따라서 삭제를 진행하시고, 관련 페이지가 모두 검색에서 사라진 것을 확인하신 다음 robots.txt 등을 설정하실 것을 추천드립니다.

### Q. 제가 소유하고 있는 사이트의 특정 페이지만 검색결과에 제외하고 싶습니다.

특정 페이지만 검색결과에 제외 시키고 싶을경우, 제외 시키고자 하는 페이지의 `<head>` 태그안에 아래와 같은 태그를 삽입합니다.

`<head><meta name="robots" content="noindex"></head>`

네이버 검색 로봇이 페이지 재방문시 해당 태그를 인식하여 색인에서 제외 합니다.  
조금 더 빠른 검색 로봇 방문을 원하신다면 웹마스터도구를 활용하여 수집요청이 가능합니다.

### Q. 접속할수 없는 페이지를 네이버 검색결과에 제외하고 싶습니다.

페이지가 삭제가 되었다면, 삭제된 내용이 검색결과에 반영되기 까지 조금만 기다려 주세요. 일정 시간이 지나면 업데이트 과정을 거쳐 자동으로 검색결과에서 제외됩니다.

웹마스터도구 소유확인을 마친 사이트라면, 웹마스터도구를 활용하여 페이지 제외요청이 가능합니다. 웹마스터도구 로그인 후, **요청 > 웹페이지 검색 제외** 메뉴를 통해 하위 주소를 입력하고 확인 버튼을 누릅니다.

단, 원본을 삭제하지 않거나, 검색 수집 제한을 하지않은 페이지는 검색결과에 다시 노출될수 있으니 유의해 주세요!

### Q. 웹사이트 운영자가 연락이 되지 않습니다. 더이상 운영하지 않는 사이트는 어떻게 네이버 검색결과에 제외시키나요?

웹사이트 운영자 (웹마스터)가 협조가 어려운 경우 사이트 호스팅 업체 등에 문의 해볼 수 있습니다.  
대부분의 사이트 호스팅 업체는 [https://후이즈검색.한국/](https://xn--c79as89aj0e29b77z.xn--3e0b707e/) 과 같이 whois 정보를 찾을수 있는곳에서 사이트 URL를 검색하면, 호스팅업체의 연락처와 같은 정보를 알아낼수 있습니다.
