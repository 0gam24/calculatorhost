# 검색노출 제목 및 설명문 갱신

### Q. 네이버 웹 검색에서 노출되고 있는 정보가 잘못되었어요

웹 검색에 노출되는 정보는 **검색 로봇에 의해 자동으로 수집 및 분석된 정보이므로 네이버에서 임의로 수정할 수 없습니다.**

따라서 사이트 운영자가 직접 정보를 수정해야 변경된 내용이 반영될 수 있습니다.  
만일 해당 사이트의 운영자라면, 아래 도움말을 참고하여 직접 수정해 주시기 바랍니다.

▶ [페이지 제목 및 설명 마크업 가이드 바로가기](<../04-HTML 마크업/02-콘텐츠 마크업.md>)

특히 로봇 수집이 전부 또는 일부 불가하거나 원활하지 않은 경우, 정보의 갱신이나 분석이 원활하지 않을 수 있습니다. 원활한 수집이 될 수 있도록 확인도 부탁드립니다.

#### URL에 대해

검색 엔진은 수집해온 문서들을 시시각각 분석하여, 동일한 내용을 가지고 있거나, 같은 페이지로 리다이렉트되는 문서 등을 정리해 대표 페이지를 추출, 검색 결과로 제공합니다. 이 경우 사이트 운영자님이 설정하지 않은 URL이 선정될 수 있으며, 이에 대한 네이버의 임의 수정은 불가능합니다.

다만, [선호 URL을 지정](https://searchadvisor.naver.com/guide/markup-structure)하는 등으로 운영자님이 대표로 하고 싶은 URL을 참고 정보로 검색 엔진에 알려줄 수 있으며, 불필요한 리다이렉트나 중복된 문서를 운영하는 서버에서 삭제하는 것도 URL 개선에 도움이 될 수 있습니다.

#### 제목과 사이트 이름, 설명문에 대해

검색 엔진은 기본적으로 해당 문서가 보유하고 있는 제목 태그 정보와 페이지 내에 제목으로 볼 수 있는 요소들을 분석하여 이용하고 있습니다.

다만, 정책상 검색 품질 향상 및 결과를 더욱 효과적으로 노출 시키기위해, 똑같은 제목이나 설명문을 가진 페이지가 다수 발견되거나, 구체적이고 간결하지 못하다고 엔진이 판단될 수 있는 경우, 로봇 수집에 지장이 있거나 불완전한 경우 등에, 앵커 (다른 페이지에서의 언급 등) 정보나 페이지에 있는 다른 텍스트 등을 활용하여 제공하는 경우가 있습니다.

가능하시면 콘텐츠 마크업 가이드라인에 [제목과 설명문 작성 tip](<../04-HTML 마크업/02-콘텐츠 마크업.md>) 을 참고해서 설정해주시면 도움이 됩니다.

#### 이미지에 대해

검색 결과에 대한 만족도 향상을 위해 검색 엔진은 문서 내용 등을 분석해 고유성이 있는 썸네일 이미지도 함께 표시하는 것이 적합하다고 판단하는 경우, 자동적으로 이를 노출하는 경우가 있습니다.

다만, 표시되는 이미지는 엔진이 자동으로 분석한 결과이므로, 인위적 노출은 어려우며, 웹 마스터님이 선호하는 것이 노출되지 않을 수도 있습니다. 특히 이미지도 웹 문서와 별도로 로봇에 의해 수집되어야 분석/노출 가능하기에, 내용 수정시 웹 문서와 별도로 반영에 시간이 걸리는 경우가 있습니다.

추가로, 분석에 본문 등 외에 오픈그래프 이미지 태그도 대상으로 이용되는 경우가 있으며, [콘텐츠 마크업 가이드](<../04-HTML 마크업/02-콘텐츠 마크업.md>) 의 관련 항목을 참고할 수 있습니다.

### Q. OG (Open Graph) title, description 태그와 기본 title, description 태그의 다른점은 있나요? 네이버 웹 검색에서는 이 두가지 태그를 어떻게 활용하고있나요?

HTML 문서내에서 참고하는 태그가 다릅니다.  
둘다 사이트명/설명문을 뜻하며 웹페이지가 검색결과에 노출하는 제목, 설명문이 됩니다.

#### 1\. 사이트 제목

- 사이트명 : HTML 문서의 태그내에 있는 `<title>페이지 제목</title>` 태그 입니다.
- Open Graph 제목 : HTML 문서의 태그내에 있는 `<meta property="og:title" content="페이지 제목">` 메타 정보 입니다.

#### 2\. 사이트 설명

- 사이트설명 : HTML 문서의 태그내에 있는 `<meta name="description" content="페이지 설명">` 메타 정보 입니다.
- Open Graph 설명 : HTML 문서의 태그내에 있는 `<meta property="og:description" content="페이지 설명">` 메타 정보 입니다.

네이버 웹 검색에서는 사이트 제목, 설명이 Open Graph 제목, Open Graph 설명과 다른 경우, 색인 과정에서 **검색에 최적화된 값을 선택하여** 검색 결과에 반영됩니다.
