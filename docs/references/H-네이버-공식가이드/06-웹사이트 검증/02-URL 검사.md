# URL 검사

웹마스터도구에서 제공하는 URL 검사는 네이버 검색로봇이 사이트의 콘텐츠 수집과 색인 정보를 제공하는 도구입니다.  
추가로 URL 문서의 메타와 SEO 정보를 제공하여 최적화가 부족한 부분을 안내하고 있습니다  
색인/Meta/SEO 각 항목은 세 테이블로 분리되어 노출되고, 클릭시 상세 내용이 하단에 노출됩니다.  
`(각 항목의 데이터가 있는 경우만 테이블이 노출됩니다.)`  
![image](https://searchadvisor-phinf.pstatic.net/MjAyMjEwMjVfMjEy/MDAxNjY2Njg1NzA0MjM1.J5hRsxnwGUmzIZLATdS4WE6HwyqhQxcOuRVsUhCuCgIg.B5gYd7vymRf1cKLkmHTloZ949nK0XYva0Y0wyOxMEkog.PNG/url-inspection-1.PNG)

### 색인

URL 상태(StatusCode)와 수집 여부, 색인 가능 여부, 색인 여부를 제공합니다.  
색인 테이블을 통해 내 URL이 수집되었는지, 수집되었다면 검색에 노출될 수 있는지(색인 여부)를 확인해 볼 수 있습니다.  
우측 아이콘은 문서 상태(색인, 수집, 미수집) 여부를 색상으로 표시합니다.  
![image](https://searchadvisor-phinf.pstatic.net/MjAyMjEwMjVfNTAg/MDAxNjY2Njg1NzA5NDA1.mQvoDYa9ytQaYXbucy9Q9xCxvv-gg59jluPzx8TnLQsg.R0XHTg7dh8mBpBDhtXFi6Y5CUGfolyOg4SmQhktvpgQg.PNG/url-inspection-2.PNG)

### Meta

HTML에 삽입된 메타 정보를 제공합니다. 결과를 보고 문제가 있다면 수정해주세요.  
우측 아이콘은 제목, 설명문 추출 가능 여부를 표시합니다.  
![image](https://searchadvisor-phinf.pstatic.net/MjAyMjEwMjVfMTk4/MDAxNjY2Njg1NzE0ODcw.dgDKa__BFw6B0htdkjYE9M2VTpRbYyXqfktd6Lbvs58g.u_tMbNaCq7y5obHXJYBenMZgBp8ODwcTXdmNMlARTVEg.PNG/url-inspection-3.PNG)

### SEO

검색엔진 최적화가 필요한 요소를 확인하여 제공합니다.  
수집과 색인에 영향을 미치는 요소들이 무엇인지 확인하고 수정해주세요.  
우측 아이콘과 SEO 요소별 색상은 SEO 중요도(반드시 수정, 수정하면 좋음, 유지해도 좋음)를 색상으로 표시합니다.
