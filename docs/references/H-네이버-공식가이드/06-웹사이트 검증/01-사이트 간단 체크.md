# 사이트 간단 체크

서치 어드바이저에서 제공하는 [사이트 간단체크](https://searchadvisor.naver.com/tools/sitecheck)는 웹마스터도구에 사이트를 등록하기 이전에 사이트의 검색 최적화 여부(SEO)를 체크할 수 있는 진단도구입니다. 사이트를 직접 개발하기 어려운 상황이라면 SEO를 준수하는 템플릿을 제공하는 좋은 업체를 이용하는 것도 한 방법입니다. 사이트 구축이전에 업체의 포트폴리오를 받아서 간단체크를 실행해 보세요. 검색 최적화를 잘 준수하고 있는지를 실시간으로 확인해 볼 수 있습니다.

![사이트 간단체크](https://searchadvisor-phinf.pstatic.net/MjAxOTEwMjRfMjcw/MDAxNTcxODk1NDM0NzMx.NAE60cFrqXlzLKUITkApYmn4Bt4EA7cNJl3HDHrjFqIg.mJnG9kWBeUYJaO68-Af1oqt25klLuJx7WEQm-zy5itYg.PNG/diagnose-site-1.png)

사이트 간단체크를 실행하면 실시간으로 해당 사이트의 메인 페이지를 방문하여 HTML 분석 후 결과를 보여줍니다. 간단체크 결과 리포트를 참고하여 사이트 개발 담당자와 함께 점검해보세요.

### 검색로봇

###### 네이버 검색로봇이 사이트에 정상적으로 접근이 가능한가요?

###### 네이버 검색로봇이 사이트의 정보를 수집할 수 있나요? (robots.txt)

###### 사이트 제목, 사이트 설명이 설정되어 있나요? (title, description 메타 태그)

###### 네이버 검색이 보유하신 사이트를 검색에 반영할 수 있나요? (noindex 설정 없음)

###### 사이트 제목, 사이트 설명이 너무 길지는 않은가요? (제목은 40자, 설명은 80자 이내로 작성하기를 권장합니다.)

###### 사이트 제목, 사이트 설명에 반복되는 단어가 많이 있지는 않나요?

###### 혹시 사이트 콘텐츠와 관련이 없는 키워드가 들어가 있지 않나요?

###### 생성되는 콘텐츠를 검색에 누락 없이 반영하기 위해 웹마스터도구를 통해 rss, sitemap을 제출하였나요?

### 콘텐츠

###### 사이트가 꾸준히 관리되고 있나요?

###### 게시판 등에 스팸 게시글이 생성되지는 않았나요?

###### 사이트 내에서 주제와 관련된 나만의 콘텐츠가 게시되고 있나요?
