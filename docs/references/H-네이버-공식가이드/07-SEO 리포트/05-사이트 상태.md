# 사이트 상태

웹마스터도구에서 제공하는 사이트 상태 정보는 네이버 검색로봇이 사이트의 콘텐츠를 수집하는 과정에서 확인된 개선점을 제공하는 리포트입니다.  
사이트 수집과 색인은 정상으로 처리되나, 더 나은 사이트 품질을 위해 개선되었으면 하는 부분을 안내하고 있습니다. `(사이트 상태는 1~2일 주기로 업데이트됩니다.)`

### 보안 인증서

사이트에서 제공하는 보안 인증서에 대하여 바르게 등록되어 있는지, 신뢰할 수 있는 인증 기관에서 발급한 것인지, 인증서가 만료되지 않았는지 등 유효성을 확인합니다.

| 상태 | 메시지 | 설명 |
| --- | --- | --- |
| 정상 | 보안 인증서 정상입니다. | 문제 없음 |
| 확인필요 | 보안 인증서 확인이 필요합니다. | 일부 문서 수집시, 유효하지않은 인증서 확인 |
| 확인필요 | 서치어드바이저에 HTTPS 사이트로 등록 바랍니다. | 보안 인증서가 존재하지만, HTTPS가 아닌 URL로 사이트를 등록한 상태 |
| 문제 | 보안 인증서 문제가 발견되었습니다. | 최근 수집된 모든 문서에서 인증서 오류 확인 |
| 미적용 | 보안 인증서를 발급 받으신후 적용하셔야 합니다. | 보안 인증서 미적용 상태 |
| 확인중 | 확인중 입니다. | \- |

### HTTPS 리다이렉션

보안 인증서 적용 이후 HTTP와 HTTPS 사이트를 동시에 운영하는 경우, 리다이렉션 상태를 확인하고 안내합니다.  
네이버 로봇은 HTTP와 HTTPS중, HTTPS를 우선하여 수집하고 있습니다. 리다이렉션없이 두 프로토콜로 사이트를 운영한다면 중복 수집 및 보안 문제 등으로 불이익을 받을 수 있습니다.  
보안 인증서를 적용했다면, HTTPS 사이트로 운영해주시기 바랍니다.  
아래 리포트 결과에 따라 HTTPS 리다이렉션 부분을 점검해 주세요.

| 상태 | 메시지 | 설명 |
| --- | --- | --- |
| 정상 | 정상입니다. | HTTP 사이트로 접속시 HTTPS 사이트로 리다이렉션 되는 경우 |
| 정상 | 네이버 검색에서는 HTTPS 통해 페이지를 제공하는것을 권장합니다. | HTTP, HTTPS 사이트 둘다 접속 되지만, HTTP로 접속시 HTTPS로 리다이렉션 안 되는 경우 |
| 확인필요 | 네이버 검색에서는 HTTPS 통해 페이지를 제공하는것을 권장합니다. | HTTPS로 접속시 HTTP 사이트로 리다이렉션되거나 HTTPS 접속이 불안정한 경우 |
| 확인중 | 확인중 입니다. | \- |

### 사이트맵

사이트맵 등록 및 상태를 확인하고 안내합니다. 아래 리포트 결과에 따라 사이트맵을 점검해주세요.

| 상태 | 메시지 | 설명 |
| --- | --- | --- |
| 정상 | 사이트맵이 등록되었습니다. | 서치어드바이저를 통해 정상 등록된 경우 |
| 정상 | robotx.txt에 사이트맵이 확인됐습니다. | 서치어드바이저로 등록하지 않았지만, robots.txt에서 사이트맵이 확인된 경우 |
| 미등록 | 사이트맵을 제출해주세요 | 서치어드바이저로 등록하지 않았거나 robots.txt에도 확인되지 않은 경우 |
| 확인필요 | 사이트맵을 찾을 수 없습니다. | 서치어드바이저를 통해 등록했으나, 사이트맵 접속이 안 되는 경우(200 ok가 아님) |
| 확인중 | 확인중 입니다. | \- |
