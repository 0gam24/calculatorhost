## 3. 금융 공식

### 3-1. 대출 상환 (원리금균등)
```
월상환액 = P * r(1+r)^n / ((1+r)^n - 1)
  P = 원금
  r = 월이자율 (연이자율 / 12)
  n = 개월수
```

### 3-2. 단리
```
이자 = 원금 × 연이율 × 년수
```

### 3-3. 복리 (월 복리)
```
원리금 = 원금 × (1 + 월이율)^개월
```

### 3-4. DSR
```
DSR = (모든 대출의 연간 원리금 상환액) / 연소득 × 100
```

### 3-5. LTV
```
LTV = 대출액 / 담보가치 × 100
```

### 3-6. 임대수익률
```
연수익률 = (연월세 × 12 - 제경비) / 실투자금 × 100
  실투자금 = 매매가 - 보증금 + 취득부대비용
```
