법적근거: 지방세법 §127·§137·§151

**세율 (cc당)**
| 배기량 | 세율 | 근거 |
|---|---|---|
| 1,000cc 이하 | 80원/cc | 지방세법 §127 |
| 1,001~1,600cc | 140원/cc | 지방세법 §127 |
| 1,601cc 이상 | 200원/cc | 지방세법 §127 |

**차령경감** (지방세법 §137②)
- 3년차부터 시작
- 연 5% (최대 50%)
- 공식: `(차령 - 2) × 5%` (단, ≤ 50%)

**지방교육세** (지방세법 §151)
- 자동차세의 30%

**연납 할인** (1월 일시납부, 지방세법 시행령 §125)
- **공제율: 5% (2025.1.1~ 유지, 2026 현행)**
  - 산정식: 할인액 = 연간 총액 × (선납일수 / 365) × 5%
  - 선납일수: 납부기한 다음날 ~ 12월 31일 (신청월별로 다름)
  - 신청월별 실효율:
    - 1월 신청(351일): ~4.81%
    - 3월 신청(273일): ~3.74%
    - 6월 신청(214일): ~2.93%
    - 9월 신청(122일): ~1.67%
  - 신청 기간: 매년 1월 1일 ~ 31일만 가능
  - 검증: 경향신문 2025-01-12, 서초구청·성동구청 2025년 공지, 2026-06-02 확정
  - **상세**: @see ADR-014

## 3. 금융 공식

### 3-1. 대출 상환 (원리금균등)
```
월상환액 = P * r(1+r)^n / ((1+r)^n - 1)
  P = 원금
  r = 월이자율 (연이자율 / 12)
  n = 개월수
```

### 3-2. 단리
```
이자 = 원금 × 연이율 × 년수
```

### 3-3. 복리 (월 복리)
```
원리금 = 원금 × (1 + 월이율)^개월
```

### 3-4. DSR
```
DSR = (모든 대출의 연간 원리금 상환액) / 연소득 × 100
```

### 3-5. LTV
```
LTV = 대출액 / 담보가치 × 100
```

### 3-6. 임대수익률
```
연수익률 = (연월세 × 12 - 제경비) / 실투자금 × 100
  실투자금 = 매매가 - 보증금 + 취득부대비용
```
