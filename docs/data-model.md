### 2-8. 자동차세 (비영업용 승용차, 2026 기준)
법적근거: 지방세법 §127(세율)·§128(납기·징수)·§151(지방교육세)·시행령 §125(연납)

> ⚠️ **차령경감은 §137②가 아니라 §127①제2호** (2026-06-17 1차출처 교정, law.go.kr·casenote). §137은 자동차 주행분 자동차세 영역으로 소유분 차령경감과 무관. 과거 문서·코드에 차령경감 §137② 인용이 남아 있으면 즉시 §127①제2호로 정정.

**세율 — 배기량 과세 (cc당, 지방세법 §127①제1호)**
| 배기량 | 세율 | 근거 |
|---|---|---|
| 1,000cc 이하 | 80원/cc | 지방세법 §127①1호 |
| 1,001~1,600cc | 140원/cc | 지방세법 §127①1호 |
| 1,601cc 이상 | 200원/cc | 지방세법 §127①1호 |

**세율 — 그 밖의 승용자동차 정액 (전기차·수소차 등 배기량 없는 차, 지방세법 §127①제3호)**
| 구분 | 자동차세 | 지방교육세(30%) | 연 총액 |
|---|---|---|---|
| 비영업용 | 100,000원 | 30,000원 | **130,000원** |
| 영업용 | 20,000원 | 6,000원 | **26,000원** |

- 전기차(BEV)·수소전기차(FCEV)는 배기량이 없어 §127①제3호 "그 밖의 승용자동차"로 분류 → **정액 과세** (차량가·배터리 용량 무관).
- **차령경감 미적용**: §127①제2호 경감액 산정 기준이 "제1호에 따른 연세액"(=배기량 과세차)이므로 정액 과세되는 제3호(전기차)는 경감 대상 아님 → 차령 무관 매년 13만원 정액. 하이브리드(HEV/PHEV)는 내연기관 배기량이 있어 §127①제1호 cc 과세 + 차령경감 적용(전기차와 구분).
- 검증: law.go.kr·casenote 지방세법 §127①제2·3호 원문, 2026-06-17 확정. 가이드: `/guide/electric-vehicle-tax-2026/`

**차령경감** (지방세법 §127①제2호 — 배기량 과세차만)
- 3년차부터 시작
- 연 5% (최대 50%, (차령−2)×5% ≥ 50% 되는 12년차부터 50% 상한)
- 공식: `(차령 - 2) × 5%` (단, ≤ 50%)
- ※ 정액 과세 전기차·수소차에는 적용되지 않음

**지방교육세** (지방세법 §151)
- 자동차세의 30%

**연납 할인** (1월 일시납부, 지방세법 시행령 §125)
- **공제율: 5% (2025.1.1~ 유지, 2026 현행)**
  - 산정식: 할인액 = 연간 총액 × (선납일수 / 365) × 5%
  - 선납일수: 납부기한 다음날 ~ 12월 31일 (신청월별로 다름)
  - 신청월별 실효율:
    - 1월 신청(351일): ~4.81%
    - 3월 신청(273일): ~3.74%
    - 6월 신청(214일): ~2.93%
    - 9월 신청(122일): ~1.67%
  - 신청 기간: 매년 1월 1일 ~ 31일만 가능
  - 검증: 경향신문 2025-01-12, 서초구청·성동구청 2025년 공지, 2026-06-02 확정
  - **상세**: @see ADR-014

### 2-9. 퇴직소득세 (연분연승법, 2026 기준)
법적근거: **소득세법 §48 (퇴직소득공제), §55② (연분연승 산출세액)**

> ⚠️ **§148의4는 실재하지 않는 가짜 조항** (2026-06-15 1차출처 교정). 퇴직소득공제·근속연수공제·환산급여공제는 모두 **§48**, 연분연승 산출세액(과세표준×세율÷12×근속연수)은 **§55②**. 원천징수 시기·방법은 §146. 과거 코드·문서에 §148의4가 남아 있으면 즉시 정정.

**계산 순서 (연분연승법)**
1. 퇴직소득금액 = 퇴직급여 − 비과세
2. 환산급여 = (퇴직소득금액 − 근속연수공제) ÷ 근속연수 × 12  *(§48)*
3. 과세표준 = 환산급여 − 환산급여공제  *(§48)*
4. 환산산출세액 = 과세표준 × 기본세율(§55①)
5. 퇴직소득산출세액 = 환산산출세액 ÷ 12 × 근속연수  *(§55②)*

**근속연수공제** (§48, 2023 개정·2026 유지)
| 근속연수 | 공제액 |
|---|---|
| 5년 이하 | 100만 × 근속연수 |
| 5년 초과~10년 | 500만 + (근속−5) × 200만 |
| 10년 초과~20년 | 1,500만 + (근속−10) × 250만 |
| 20년 초과 | 4,000만 + (근속−20) × 300만 |

**환산급여공제** (§48)
| 환산급여 | 공제액 |
|---|---|
| 800만 이하 | 전액 |
| 800만~7,000만 | 800만 + 초과분 × 60% |
| 7,000만~1억 | 4,520만 + 초과분 × 55% |
| 1억~3억 | 6,170만 + 초과분 × 45% |
| 3억 초과 | 1억 5,170만 + 초과분 × 35% |

- 지방소득세: 퇴직소득세의 10% (지방세법 §103)
- 상수: `src/lib/constants/tax-rates-2026.ts` `SERVICE_YEARS_DEDUCTION_BRACKETS` / `CONVERTED_SALARY_DEDUCTION_BRACKETS`
- 검증: law.go.kr 소득세법 제48조·제55조, 국세청 퇴직소득세 계산방법, 2026-06-15 확정

## 3. 금융 공식

### 3-1. 대출 상환 (원리금균등)
```
월상환액 = P * r(1+r)^n / ((1+r)^n - 1)
  P = 원금
  r = 월이자율 (연이자율 / 12)
  n = 개월수
```

### 3-2. 단리
```
이자 = 원금 × 연이율 × 년수
```

### 3-3. 복리 (월 복리)
```
원리금 = 원금 × (1 + 월이율)^개월
```

### 3-4. DSR
```
DSR = (모든 대출의 연간 원리금 상환액) / 연소득 × 100
```

### 3-5. LTV
```
LTV = 대출액 / 담보가치 × 100
```

### 3-6. 임대수익률
```
연수익률 = (연월세 × 12 - 제경비) / 실투자금 × 100
  실투자금 = 매매가 - 보증금 + 취득부대비용
```
