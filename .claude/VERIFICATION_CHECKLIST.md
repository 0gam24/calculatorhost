> **목적**: 새 가이드·콘텐츠 포스트의 시뮬레이션·세금 계산 사례가 법조항 기반인지 확인.
> **운영 모드**: content-writer → calc-logic-verifier → merge review (메인 스레드)
> **Last Updated**: 2026-05-14

## 핵심 원칙

1. **법조항 근거 없이 세율값 승인 금지**
2. **누진세 = 누진공제 필수** (덩어리식 계산 금지)
3. **경계값 테스트 필수** (세율 구간 경계에서 상/하 둘 다 검증)
4. **공식 계산기 교차검증** (국세청 또는 금감원)
5. **단위·반올림 규칙 명시** (10원 절사 등)

## 체크리스트 (신규 가이드/콘텐츠 발행 전)

### A. 종합소득세 (근로소득 포함)

**법적근거**: 소득세법 §55 (세율표) + §59의2 (근로소득공제)

- [ ] **누진공제 적용 확인**
  - 1400만 초과 5000만 이하: 126만 공제 (15% 구간)
  - 5000만 초과 8800만 이하: 576만 공제 (24% 구간)
  - 1.5억 초과: 1,544만 공제 (35% 구간)
  - 식별 오류: "5000만 * 15% = 750만" (누진공제 미적용) → RED
  - 정정: "(5000만 - 1400만) * 15% + 1400만 * 6% - 126만 = 680만" (누진공제+누진 적용) → GREEN

- [ ] **근로소득공제 적용** (특정 급여 구간만)
  - 500만 이하: 70% 공제
  - 500만~1500만: 350만 + 초과분 40%
  - 1500만~4500만: 750만 + 초과분 15%
  - 사례: 연봉 5000만 → 근로소득공제 750만 + 초과분(3500만)*15% = 750만 + 525만 = 1275만

- [ ] **비과세 근로소득 확인**
  - 월 20만 식대 비과세 (연 240만)
  - 자가운전보조금 월 20만 비과세 (연 240만)
  - 누락 시 실수령액 -1.2% 이상 오차 발생

- [ ] **4대보험 공제 순서**
  - 순서: 총급여 → 비과세 공제 → 4대보험 → 근로소득공제
  - 오류 패턴: 비과세 후 4대보험을 차감하지 않음 → 사회보험료 공제 누락

- [ ] **부양가족 공제 (기본공제)**
  - 1인당 150만원 (소득 100만 이하만 해당)
  - 자녀 2명 기준: 150만 * 2 = 300만 공제
  - 배우자 1 + 부모 1 + 자녀 2 = 600만 공제 (최대)

- [ ] **자녀세액공제**
  - 1-2인: 1인당 15만원
  - 3인 이상: 1인당 30만원
  - 사례: 자녀 1명 = 15만원 / 자녀 3명 = 30만 * 3 = 90만원

### B. 양도소득세 (주택)

**법적근거**: 소득세법 §92-§118, 시행령 §154 (일시적 2주택)

- [ ] **기본공제 250만 원 차감**
  - 양도소득금액 - 250만 = 과세표준
  - 오류: 기본공제 미적용 → 세액 3-4% 과다 계산

- [ ] **1세대1주택 비과세 조건 (모두 충족)**
  - ① 보유 2년 이상 (소유권 등기일 기준)
  - ② 양도가액 12억원 이하 (초과분만 과세)
  - ③ 1세대 1채만 해당 (부부합산)
  - 식별 오류: "12억 한도 초과 시 전액 과세" → RED
  - 정정: "12억 초과분만 과세" → GREEN
    - 사례: 양도가 15억 → 과세표준 = (15억 - 12억) = 3억 (비례 적용)

- [ ] **일시적 2주택 비과세 (신주택 취득 후 구주택 양도)**
  - 신주택 취득일로부터 2년(조정지역 6개월) 내 구주택 양도 시 1세대1주택으로 인정
  - 소득세법 §94②
  - 기한 경과 시 2주택 중과세 대상 (추가 세율 20-30%p)

- [ ] **조정지역 가중세**
  - 조정 + 2주택: 세율 +20%p
  - 조정 + 3주택↑: 세율 +30%p
  - 적용 시점: 양도하는 시점의 조정지역 지정 여부 (취득 시점 아님)

- [ ] **장기보유특별공제**
  - 1세대1주택: 4~10년 → 4~40% 공제 (1년마다 누적)
  - 일반: 6~30% 공제
  - 공제 대상: 취득가 × 공제율

- [ ] **누진세율 적용 (양도소득세도 누진)**
  - 과세표준이 다양한 구간에 해당하면 § 55 종합소득세 세율 적용
  - 사례: 과세표준 3억원 → 38% 구간 (누진공제 1,994만)
  - 검산: 3억 * 38% - 1,994만 = 1,144만 - 1,994만 = 계산 오류 (음수)
  - 정정: (1.5억 이상) 구간이므로 계산 재검토 필요

- [ ] **경계값 테스트 (12억 한도)**
  ```
  사례 1 (하한): 양도가 11.5억
  → 비과세, 세금 0원
  
  사례 2 (정확히 경계): 양도가 12억 
  → 비과세, 세금 0원
  
  사례 3 (초과): 양도가 12.5억
  → 과세표준 = (12.5억 - 12억) = 5000만
  → 세율 15% 적용 → 세액 = 5000만 * 15% - 126만 = 636만
  ```

### C. 취득세 (주택)

**법적근거**: 부가가치세법 제11조의2, 지방세법 §3

- [ ] **1주택자 세율 (가격별 구간)**
  - 3억 이하: 1.0%
  - 3~9억: 1.5%
  - 9억 초과: 3.0%
  - 오류 예시: "취득세 3%" (구간 미적용) → 모든 구간에 적용하면 과다 계산

- [ ] **2주택 이상 (조정지역)**
  - 조정 + 2주택: 8.0%
  - 조정 + 3주택↑: 12.0%
  - 법령: 지방세법 § 11

- [ ] **지방교육세 (취득세의 10%)**
  - 취득세 * 10% (별도 징수)
  - 총세액에 포함 명시 필수

- [ ] **85㎡ 초과 농특세 (+0.2%)**
  - 85제곱미터 초과 주택에만 적용
  - 합산: 취득세 + 농특세 + 지방교육세 = 총 취득세

- [ ] **경계값 테스트 (3억, 9억)**
  ```
  사례 1: 구입가 2.9억 → 취득세 2,900만 * 1.0% = 29만
  사례 2: 구입가 3.0억 → 취득세 3,000만 * 1.5% = 45만
  사례 3: 구입가 8.9억 → 취득세 8,900만 * 1.5% = 133.5만
  사례 4: 구입가 9.0억 → 취득세 9,000만 * 3.0% = 270만
  ```

### D. 재산세

**법적근거**: 지방세법 § 3 (재산세)

- [ ] **누진공제 적용 (4단계)**
  - ~6000만: 0.1%, 누진공제 0
  - ~1.5억: 0.15%, 누진공제 3만
  - ~3억: 0.25%, 누진공제 18만
  - 3억↑: 0.4%, 누진공제 63만
  - 공식: 과표 * 세율 - 누진공제

- [ ] **1세대1주택 특례 (공시 9억 이하)**
  - 세율 절반 수준 (상기 세율의 약 50%)
  - 공시가격 기준 적용 (실제 거래가 아님)

- [ ] **경계값 테스트 (6000만, 1.5억, 3억)**
  ```
  사례: 과표 1.5억
  → 0.15% 구간 = 1.5억 * 0.15% - 3만 = 225만 - 3만 = 222만
  ```

### E. 증여세·상속세

**법적근거**: 소득세법 §55 (세율), 증여세법 (공제)

- [ ] **누진세율 (5단계)**
  - 1억 이하: 10%
  - ~5억: 20% (누진공제 1000만)
  - ~10억: 30% (누진공제 6000만)
  - ~30억: 40% (누진공제 1.6억)
  - 30억 초과: 50% (누진공제 4.6억)

- [ ] **누진공제 적용**
  - 예: 증여가 3억원 → 20% 구간 적용
  - 세액 = 3억 * 20% - 1000만 = 6000만 - 1000만 = 5000만

- [ ] **증여재산공제 (10년 단위)**
  - 배우자: 6억원
  - 성년 직계비속: 5000만원
  - 미성년: 2000만원
  - 차감 순서: 증여가액 - 공제 = 과세표준

- [ ] **경계값 테스트 (1억, 5억, 10억, 30억)**
  ```
  사례 1: 자녀 증여 1.2억
  → 공제 5000만 → 과세표준 = 1.2억 - 5000만 = 7000만
  → 세율 10% (1억 이하) → 세액 = 7000만 * 10% = 700만
  
  사례 2: 자녀 증여 5.5억
  → 공제 5000만 → 과세표준 = 5억
  → 세율 20% (5억 구간 상한) → 세액 = 5억 * 20% - 1000만 = 9000만
  ```

### F. 금융 공식 (대출·예금·적금)

- [ ] **원리금균등 상환 공식 정확성**
  ```
  월상환액 = P × r(1+r)^n / ((1+r)^n - 1)
  P = 원금, r = 월이율, n = 개월수
  ```
  - 검증: 계산기 결과 vs 은행권 공식 계산기 대조

- [ ] **단리 vs 복리 구분**
  - 단리: 이자 = 원금 × 연이율 × 년수
  - 복리(월): 원리금 = 원금 × (1 + 월이율)^개월
  - 사례 혼동 방지

- [ ] **DSR 계산**
  ```
  DSR = (모든 대출 연간 상환액) / 연소득 × 100
  ```
  - 주담대 + 전세대출 모두 포함
  - 월 상환액 × 12 계산 확인

- [ ] **경계값 테스트**
  - 원금 변화 10배 → 결과 선형 증가 확인
  - 금리 ±1% → 월상환액 변화 확인

## 흔한 오차 패턴 (Red Flags)

### 1. 누진공제 누락
```
오류: 과세표준 3000만 × 15% = 450만 (GREEN 처럼 보임)
실제: (5000만 상한이므로) 3000만 × 15% - 126만 = 324만
```

### 2. 기본공제 미적용
```
오류: 양도차익 5억 → 과세표준 5억 (250만 미적용)
정정: 과세표준 = 5억 - 250만 = 4.975억
```

### 3. 1세대1주택 12억 한도 전체 비과세 오류
```
오류: "양도가 15억은 비과세 불가능"
정정: "12억까지 비과세, 초과분 3억만 과세"
```

### 4. 구간 경계 반올림
```
예시: 과세표준 4,999만 → 15% 구간? 아니면 6%?
정정: "~5,000만원" = 5,000만 이하 → 15% 구간
```

### 5. 비례 과세 오류 (1세대1주택)
```
오류: 양도가 15억, 비례 계산 = 12억/15억 = 80%만 비과세?
정정: 과세표준 = (15억 - 12억) = 3억만 과세
```

## 검증 워크플로우

### Phase 1: content-writer (초안 작성)
```
1. 포스트에 시뮬레이션 포함 시 이 체크리스트 자가 검수
2. 법조항 명시 (§55, §92 등)
3. REFERENCE.md SSoT 값과 대조
4. 간단한 경계값 케이스 2-3개 자체 검산
5. calc-logic-verifier에 PR/검토 요청
```

### Phase 2: calc-logic-verifier (감사)
```
1. REFERENCE.md 해당 섹션 읽음 (이 체크리스트 조항별)
2. 계산 공식 · 누진공제 · 경계값 재검증
3. 국세청 간이계산기 또는 공식 예시와 교차검증
4. 오류 시: RED 마크 + 정정 요청
5. PASS 시: GREEN 마크 + approve 또는 comment
```

### Phase 3: 메인 스레드 (merge 전)
```
1. calc-logic-verifier 승인 여부 확인
2. 단위·반올림 설명 명시되었는지 재확인
3. YMYL 성격 콘텐츠면 법조항 원문 링크 필수 확인
4. merge 진행 또는 재요청
```
