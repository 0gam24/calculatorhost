# AdSense 자기 클릭 방지 영구 규칙

> **상태**: 영구 운영 규칙 (변경 금지)
> **사유**: AdSense 정책 §2 위반 시 계정 영구 정지 + 같은 명의 모든 계정 차단
> **참조**: https://support.google.com/adsense/answer/3027999

## 1. 절대 금지 행위

### A. 운영자 본인의 광고 상호작용
- ❌ **자신의 사이트에서 광고 클릭** (실수든 의도든 모두)
- ❌ 광고 노출 빈도 확인 명목으로 페이지 새로고침 반복
- ❌ "내 광고가 어떻게 보이는지" 확인 위한 클릭
- ❌ 모바일/데스크톱 광고 차이 확인 명목 클릭
- ❌ 결제 직전 광고 클릭으로 RPM 확인

### B. 가족·지인·동료의 클릭 유도
- ❌ "우리 사이트 봐줘" SNS 게시물 (광고 노출 페이지 직링크)
- ❌ 카카오톡 단톡방 / 디스코드 / 슬랙에 페이지 공유 (광고 노출 시점)
- ❌ "광고 클릭해주면 고마워" 직간접 암시
- ❌ 가족·친구·동료에게 사이트 베타 테스트 요청 시 광고 영역 명시

### C. 트래픽 부정 조작
- ❌ 봇 트래픽 구매 / 클릭 봇 배포
- ❌ "클릭 교환" 프로그램 참여 (다른 운영자와 상호 클릭)
- ❌ Mechanical Turk / 크라우드워크 통한 인위적 트래픽
- ❌ VPN/프록시 통한 다중 IP 클릭 시도

## 3. 모니터링 (주간 1회)

### GA4 + AdSense 대조 체크
| 지표 | 정상 범위 | 경고 임계값 | 위험 |
|---|---|---|---|
| 광고 CTR | 1~3% | >3% | >5% (즉시 조사) |
| 평균 세션 시간 | >30초 | <15초 | <5초 (봇 의심) |
| 한국 IP 비율 | >80% | 60~80% | <50% (봇 의심) |
| 동일 IP 반복 클릭 | 없음 | 1~2회 | 3회+ (즉시 차단) |

## 4. 위반 발견 시 즉시 대응

### 시나리오: 운영자 또는 가족이 실수로 클릭
1. **즉시 AdSense 대시보드 → 정책 센터 → "무효 트래픽 보고"** 자진 신고
2. 신고 내용: 발생 일시, IP, 클릭 횟수, 의도성 여부
3. 자진 신고 시 경고로 끝날 수 있음 (계정 정지 회피)
4. 절대 은폐 시도 금지 — Google 의 자동 탐지 시스템이 더 정확

### 시나리오: AdSense 경고 메일 수신
1. 메일 원문 정독 (보통 위반 페이지·시간대 명시됨)
2. 위반 패턴 분석 (어떤 IP? 어떤 페이지?)
3. AdSense 정책 센터에서 상세 위반 사항 확인
4. 7일 이내 개선 보고서 제출 (해당 IP 차단·콘텐츠 수정 등)

## 5. 팀 합류 시 의무 절차

새로운 협업자(개발자·디자이너·콘텐츠 작성자)가 calculatorhost 작업에 참여 시:
1. 이 문서를 **읽고 이해했음** 명시 확인
2. 본인의 접속용 시크릿 창 환경 설정
3. 자기 클릭 의도 없어도 **무심한 광고 클릭 위험성** 인지
4. SNS 에서 사이트 공유 시 광고 노출 페이지 직링크 금지
