# Korean Tax Rates Skill

## 워크플로우

### 상황 1: 계산 공식 정확성 검증
1. 대상 함수 Read (`src/lib/tax/*.ts`)
2. `REFERENCE.md` 해당 세목 섹션 비교
3. 세율/공제/누진구조/한도 전부 확인
4. 단위 테스트 케이스로 실제 값 검증 제안

### 상황 2: 새 계산기 공식 설계
1. 국세청/행안부 공식 세율표 WebFetch
2. `REFERENCE.md`에 추가 (업데이트 필요 시)
3. 순수 함수 구조 설계 제안

### 상황 3: 세법 개정 반영
1. 기획재정부 보도자료 조사
2. 변경 세율 → REFERENCE.md 업데이트
3. `src/lib/constants/tax-rates-{year}.ts` 신규 파일 생성 제안
4. 기존 constants 유지 (과거 거래 계산용)

## 통과 기준
- 모든 세율 값에 법조항 근거 주석
- 누진세 구조 정확 구현 (단계별 초과분)
- 공제액 한도 적용
- 가산세 로직 포함
