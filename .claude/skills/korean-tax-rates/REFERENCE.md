## §1. 근로소득세 (종합소득세 - 근로소득)

### 법적 근거
- 소득세법 §55 (세율표)
- 소득세법 §59의2 (근로소득공제)
- 소득세법 §50-§51 (기본공제·추가공제)

### 2026년 과세표준 구간 (8단계 누진)
| 과세표준 | 세율 | 누진공제 |
|---|---|---|
| ~ 1,400만원 | 6% | 0 |
| ~ 5,000만원 | 15% | 126만원 |
| ~ 8,800만원 | 24% | 576만원 |
| ~ 1.5억원 | 35% | 1,544만원 |
| ~ 3억원 | 38% | 1,994만원 |
| ~ 5억원 | 40% | 2,594만원 |
| ~ 10억원 | 42% | 3,594만원 |
| 10억원 초과 | 45% | 6,594만원 |

### 근로소득공제
| 총급여 | 공제액 |
|---|---|
| ~ 500만원 | 70% |
| 500만~1500만 | 350만 + 초과분 40% |
| 1500만~4500만 | 750만 + 초과분 15% |
| 4500만~1억 | 1200만 + 초과분 5% |
| 1억 초과 | 1475만 + 초과분 2% (한도 2000만) |

### 4대보험 요율 (2026년 기준, 확인 필요)
- 국민연금: 9% (근로자 4.5%)
- 건강보험: 7.09% (근로자 3.545%)
- 장기요양: 건강보험의 12.95% (근로자 부담분)
- 고용보험: 1.8% (근로자 0.9%)

### 지방소득세
- 소득세의 10%

### 구현 주의
- 비과세 근로소득 공제 (월 20만원 식대, 자가운전보조금 월 20만원 등)
- 부양가족 공제: 1인당 150만원
- 자녀세액공제: 1인 15만원, 3인부터 각 30만원

## §2. 양도소득세

### 법적 근거
- 소득세법 §92-§118
- 소득세법 시행령 §154 (일시적 2주택 비과세)

### 세율 (주택, 2026년)
| 구분 | 세율 |
|---|---|
| 기본 (1-2년 보유) | 40% |
| 2년 이상 보유 | 기본세율(§55 누진) |
| 조정지역 + 2주택 | 기본 + 20%p |
| 조정지역 + 3주택 이상 | 기본 + 30%p |
| 분양권 (1년 미만) | 70% |
| 분양권 (1년 이상) | 60% |

### 1세대 1주택 비과세
- 보유 2년 이상 (조정지역 취득 시 거주 2년 추가)
- 실거래가 12억 이하 비과세, 초과분만 과세
- 일시적 2주택: 신규 취득 3년 내 종전 주택 처분 시 비과세

### 장기보유특별공제
- 1세대 1주택 거주/보유 각 4%~40%
- 일반: 6%~30%

### 필요경비
- 취득가액 + 자본적 지출 + 양도비용

### 공제
- 기본공제 250만원/연

### 계산 흐름
```
양도가액 - 취득가액 - 필요경비 = 양도차익
양도차익 - 장기보유특별공제 = 양도소득금액
양도소득금액 - 기본공제 250만 = 과세표준
과세표준 × 세율 - 누진공제 = 산출세액
```

## §3. 취득세

### 법적 근거
- 지방세법 §10-§17
- 시행령 §22~

### 주택 취득세 (2026년)
| 구분 | 1주택자 | 2주택자 | 3주택자 | 4주택이상/법인 |
|---|---|---|---|---|
| 6억 이하 | 1.0% | 8% (조정) / 1.0% (비조정) | 12% | 12% |
| 6억~9억 | 1.01~2.99% | 8% / 1~3% | 12% | 12% |
| 9억 초과 | 3.0% | 8% / 3% | 12% | 12% |
| 85㎡ 초과 | +농특세 0.2% | +농특세 0.2% | +농특세 1% | +농특세 1% |

### 생애최초 주택 감면
- 200만원 한도 감면 (조건 충족 시)

### 부가세
- **농어촌특별세 (농특세)**: 전용 85㎡ 초과 시 0.2% (일반), 중과 시 1%
- **지방교육세**: 취득세의 10% (기본 세율 기준)

### 과세표준
- 유상거래: 실거래가
- 무상(증여·상속): 시가 or 시가표준액

## §10. 구현 체크리스트

### 모든 세금 계산 함수 필수
- [ ] 순수 함수 (no side effects)
- [ ] 과세표준 먼저 계산 → 세율 적용
- [ ] 누진세는 단계별 초과분 방식 (덩어리 곱하기 금지)
- [ ] 누진공제 적용 후 산출세액
- [ ] 신고세액공제 (해당 시)
- [ ] 반올림 규칙: 10원 단위 절사
- [ ] 최소/최대 한도 적용
- [ ] 법조항 근거 주석 (`// §55 소득세법`)

### 테스트 필수
- [ ] 경계값 (구간 끝)
- [ ] 0원, 음수, 최대값
- [ ] 특례 조건 각각 (1주택자, 조정지역, 일시적 2주택 등)
- [ ] 실제 국세청 간이계산기 결과와 교차 검증
