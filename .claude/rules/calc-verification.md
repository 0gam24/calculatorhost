## 빠른 체크 (신규 포스트 작성 시)

### 시뮬레이션 포함 여부 판단
```
Q: 이 포스트에 세금/금리/수익률 계산 사례가 있는가?
YES → 아래 체크리스트 항목별로 자가 검수 실시
NO  → 일반 콘텐츠 규칙만 적용
```

### 누진세 계산 (가장 흔한 오류)
```
❌ 잘못된 패턴:
"과세표준 5000만원 × 15% = 750만원"

✅ 올바른 패턴:
"과세표준 5000만원 × 15% - 누진공제 126만 = 624만원"

📌 기억법: 세율 구간이 여러 개면 → 누진공제 필수
```

### 양도소득세 1세대1주택 12억 한도
```
❌ 잘못된 패턴:
"양도가 15억이면 1세대1주택 비과세 불가능"

✅ 올바른 패턴:
"양도가 15억이면 12억까지 비과세, 초과분 3억만 과세"
→ 과세표준 = (15억 - 12억) - 기본공제 250만 = 약 3억
→ 과세표준 3억 × 38% - 1,994만 ≈ 914만원
```

### 기본공제 250만 원 (양도소득세)
```
공식: 양도소득금액 - 기본공제 250만 = 과세표준

❌ 누락 시: 세액 과다 계산 (약 3~4%)
✅ 반드시 확인: 모든 양도소득세 계산에 포함
```

### 경계값 테스트 (최소 2개)
```
예: 취득세 계산
- 구입가 3억 이하: 1.0%
- 구입가 3억 초과~9억: 1.5%
- 구입가 9억 초과: 3.0%

필수 테스트:
1️⃣ 구입가 2.99억 → 1.0% 적용
2️⃣ 구입가 3.01억 → 1.5% 적용
3️⃣ 구입가 8.99억 → 1.5% 적용
4️⃣ 구입가 9.01억 → 3.0% 적용

→ 포스트에 최소 2~4개 사례 계산 제시
```

## 세목별 즉시 확인 항목

### 종합소득세 (연봉·프리랜서)
- [ ] 근로소득공제 적용 (구간별 공제액 다름)
- [ ] 누진공제 적용 (예: 5000만 초과 시 126만 공제)
- [ ] 비과세 근로소득 포함 (월 20만 식대 등)
- [ ] 4대보험 공제 포함
- [ ] 부양가족/자녀세액공제 명시

### 양도소득세
- [ ] 기본공제 250만 원 차감
- [ ] 1세대1주택 12억 한도 (초과분만 과세)
- [ ] 누진공제 적용
- [ ] 보유 기간별 세율 구분 (2년 미만 40%, 이상 누진)
- [ ] 조정지역 가중세 (+20~30%p)

### 취득세
- [ ] 가격별 세율 구간 (3억/9억 경계)
- [ ] 2주택 이상 중과세
- [ ] 지방교육세 +10% (취득세의 10%)
- [ ] 85㎡ 초과 농특세 +0.2%

### 대출·금리
- [ ] 원리금균등 공식 정확성
- [ ] 단리 vs 복리 구분
- [ ] DSR 계산 (연 상환액 / 연소득)

```

```


## 자주 틀리는 예시

### 사례 1: 누진공제 적용 오류
```
입력: 연봉 5,000만 → 과세표준 5,000만
❌ 세액 = 5,000만 × 15% = 750만
✅ 세액 = 5,000만 × 15% - 126만 = 624만
→ 오차: 126만 (20%)
```

### 사례 2: 1세대1주택 한도 오류
```
입력: 양도가 15억
❌ 비과세 불가능 → 세액 과다 계산
✅ 12억까지 비과세, 3억만 과세 → 세액 약 914만
→ 오차: 1,400만+ (원주택 전체 과세로 계산하면)
```

### 사례 3: 기본공제 누락
```
입력: 양도차익 5억
❌ 과세표준 = 5억
✅ 과세표준 = 5억 - 250만 = 4.975억
→ 오차: 약 3-4%
```

### 사례 4: 취득세 구간 오류
```
입력: 구입가 3억
❌ "1.0% 적용" → 세액 300만
✅ "1.5% 적용" → 세액 450만
→ 오차: 150만 (50%)
```

## 법조항 인용 가이드

**최소 필수 인용**:
- 종합소득세: 소득세법 §55
- 양도소득세: 소득세법 §92-§118
- 취득세: 부가가치세법 §11의2
- 재산세: 지방세법 §3
- 증여·상속세: 증여세법 (해당 조항)

**포스트에 명시 방법**:
```
"소득세법 §55에 따른 누진세율 적용..."
"국세청 공식 고시(2026)..."
```
