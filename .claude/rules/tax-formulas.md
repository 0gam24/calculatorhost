# 세금·금융 계산 로직 규칙

## 순수 함수 원칙
- 부작용 없음 (no fs, no fetch, no Date.now)
- 동일 입력 → 동일 출력
- 날짜·연도는 파라미터로 받기 (not hardcoded)

## 세율 상수 (비타협)
- `src/lib/constants/tax-rates-{year}.ts`에만 저장
- 매 값 옆 법조항 주석: `// 소득세법 §55`
- 이전 연도 상수 삭제 금지 (과거 거래 계산용)

## 누진세 구현 방식
- ❌ `과세표준 * 세율` (덩어리 계산)
- ✅ 구간별 초과분 × 해당 세율 + 누진공제

## 반올림 규칙
- 세금 결과: 10원 단위 절사 (Math.floor(x / 10) * 10)
- 비율(%)은 소수점 2자리 반올림
- 월 환산: `/ 12` 후 정수 반올림

## 필수 테스트
- 경계값 (구간 끝)
- 0원, 음수, 최대값
- 특례 조건 각각 (1주택자/조정지역/일시적2주택 등)
- 국세청 간이계산기 결과와 교차 검증

## 금지
- 매직넘버 (반드시 상수)
- 추측 세율 (법조항 근거 없으면 PR 차단)
- 동적 세율 조회 (공공 API 없으면 정적 상수)
- 연도 혼용 (ex: 2025 세율로 2026 거래 계산)
- 계산 중간 `console.log` 잔존

## 에이전트 위임
세율 값 추가/수정은 반드시 **calc-logic-verifier** 통과 후.
